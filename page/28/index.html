维尔斯：波罗的海冥界之神与生死平衡

老梵 发布于 2026-05-08 收录于 波罗的海神话

前言

维尔斯（Vėlinas，立陶宛语；拉脱维亚语为Velnias）是波罗的海神话中的冥界之神，掌管死者与地下世界。他最初是丰收之神，因为种子必须进入地下（死亡）才能发芽（重生）。基督教化过程中，维尔斯被妖魔化为"魔鬼"（veltė），但民间仍秘密崇拜他，形成了独特的双信仰文化。

维尔斯：波罗的海冥界之神与生死平衡

关键词：维尔斯、冥界之神、波罗的海神话、生死循环、丰收之神、魔鬼替身、双信仰、立陶宛

维尔斯的起源与身份

1.1 从丰收之神到冥界之神

维尔斯最初是丰收之神，掌管种子与地下世界的生命力。立陶宛农民相信，种子必须进入地下（象征死亡），才能吸收土地的营养，最终发芽生长（象征重生）。因此，维尔斯同时掌管死亡与丰收——地下世界的统治者，也是新生命的孕育者。

基督教传入后，维尔斯被妖魔化为"魔鬼"（veltė）。教会告诉农民：维尔斯是邪恶的，是诱惑者，是地狱之主。但农民并不买账——他们仍需向维尔斯祈祷，因为种子仍需进入地下才能发芽。

1.2 迪耶瓦斯的兄弟或对立面

维尔斯与天空之神迪耶瓦斯是对立又互补的关系。迪耶瓦斯代表天空、光明、永恒；维尔斯代表地下、黑暗、暂时。两者共同维持宇宙的平衡——没有地下世界，就没有生命的循环；没有死亡，就没有重生。

立陶宛民歌中，迪耶瓦斯与维尔斯有时被描绘为兄弟：一个住天上，一个住地下；一个给予生命，一个接纳死亡。这种"兄弟对立"的母题，体现了波罗的海人对生死平衡的理解。

1.3 冥界的统治者

维尔斯住在地下世界，宫殿由黑曜石与黑铁建成。他接纳所有死者，无论贫富、善恶，一视同仁。与希腊哈迪斯的冷漠、北欧赫尔的歧视不同，维尔斯对死者一视同仁——在他的宫殿里，所有人都是平等的。

立陶宛人相信，死者会回归大地母亲热米娜的怀抱，由维尔斯负责接待。墓地常设在田野边缘，象征死者回归大地，继续滋养庄稼。

神职与象征

2.1 死亡与重生

维尔斯最核心的神职是死亡与重生。他接纳死者，让他们的身体回归大地，养分滋养庄稼；他让种子"死亡"（进入地下），换来新的生命（发芽生长）。

这种"生死一体"的观念，使维尔斯成为最辩证的神祇。他不是邪恶的，而是自然的——死亡不是终结，而是新生命的开始；地下世界不是地狱，而是生命的仓库。

2.2 丰收与农耕

维尔斯最初是丰收之神，掌管种子、土壤、地下水分。农民播种时会向维尔斯祈祷，因为种子必须进入他的领域才能发芽。

立陶宛农民在播种时会说：“Vėlinas, priimk sėklą”（维尔斯，接纳种子），承认地下世界对丰收的重要性。这种对"死亡之神"的祈祷，体现了波罗的海人务实的宗教态度——不管神的名字是什么，只要能带来丰收，就值得崇拜。

2.3 象征物

维尔斯的象征物包括：

  • 黑牛：献祭给维尔斯最常用的动物。
  • 黑麦面包：象征地下世界的粮食。
  • 黑曜石：地下世界的宝石。
  • 蛇：地下世界的守卫者。
  • 蘑菇：从地下生长的生命。

所有这些象征物都是黑色的或地下的，强调维尔斯与地下世界的联系。

神话故事

3.1 维尔斯与佩尔库纳斯的对抗

维尔斯与佩尔库纳斯（雷神）是对立又互补的关系。佩尔库纳斯代表天空与雷电，维尔斯代表地下与死亡。当佩尔库纳斯用雷电劈开大地时，实际上是打开通往地下世界的门，让雨水滋润维尔斯的领域。

拉脱维亚民间传说中，佩尔库纳斯与维尔斯曾争夺热米娜（大地女神）的爱情。佩尔库纳斯代表天空的滋润，维尔斯代表地下的养分，两者共同养育大地。

3.2 维尔斯与死者的旅程

立陶宛传说中，死者会踏上前往维尔斯宫殿的旅程。他们穿过七道门、渡过七条河，最终到达地下世界。维尔斯会审视死者的生平，但不做审判——在他的宫殿里，所有人都是平等的。

这个"平等死亡"的观念，体现了波罗的海人的民主精神。无论生前是国王还是农民，死后都回归大地，接受维尔斯的平等接待。

3.3 维尔斯与双信仰

基督教传入后，维尔斯被妖魔化为"魔鬼"。但农民并不买账——他们继续向维尔斯祈祷，继续在播种时向他献祭。这种"表面基督徒，内心异教徒"的双信仰，一直持续到19世纪。

19世纪的民俗学者记录道：“农民在教堂受洗后，会偷偷来到田野，向维尔斯献祭黑牛，祈求庄稼丰收。他们不认为这有什么矛盾——上帝管天上的事，维尔斯管地下的事。”

祭祀与仪式

4.1 播种祭祀

立陶宛农民在播种时会向维尔斯献祭。他们会在田边挖一个小坑，放入黑麦面包、啤酒、奶酪，说：“Vėlinas, priimk auką”（维尔斯，接纳供品），祈求种子顺利发芽。

这种播种祭祀体现了波罗的海人的务实精神——不管神是善是恶，只要能帮上忙，就值得崇拜。

4.2 墓地选址

立陶宛人会将墓地设在田野边缘或橡树林旁，因为那里最接近维尔斯的领域。他们相信，死者回归大地后，会化作养分滋养庄稼，完成生命的循环。

考古发现，波罗的海地区的古墓地常与农田相邻，这种布局正是"生死循环"观念的体现。死者不是被"埋葬"，而是"回归"；不是"消失"，而是"转化"。

4.3 黑牛献祭

黑牛是献给维尔斯最常用的祭品。立陶宛农民会在秋收后献上一头黑牛，感谢维尔斯的恩赐，祈求来年丰收。献祭仪式在夜晚进行，因为维尔斯是地下世界的神。

这种黑牛献祭一直持续到19世纪，即使在基督教化后，农民仍秘密举行。教会称这种仪式为"魔鬼崇拜"，但农民认为这只是对土地的感恩。

前言

在印度神话体系中，杜尔迦（Durga）是最具勇猛色彩的愤怒女神，据《往世书》记载，她“十臂持兵器，骑狮斩魔王摩醯湿（Mahisha）”。作为雪山神女（Parvati）的愤怒化身，她深深影响了印度的性力崇拜、妇女力量、战争文化与艺术表现，成为“愤怒即力量”的永恒象征。

杜尔迦：雪山神女的愤怒化身

一、杜尔迦的起源与地位

1.1 雪山神女的愤怒化身

据《往世书》记载，众神无法击败魔王摩醯湿（Mahisha，水牛怪），便集合神力，从雪山神女（Parvati）的愤怒中诞生出杜尔迦。她象征“性力（Shakti）的愤怒爆发”。

1.2 斩杀魔王摩醯湿

杜尔迦最重要的功绩是斩杀魔王摩醯湿：

  • 九天九夜的激战
  • 十臂持兵器，最终用三叉戟刺穿魔王心脏
  • 拯救三界，恢复秩序（Dharma）

1.3 与迦梨（Kali）的对比

杜尔迦是“勇猛的愤怒”（有限度的愤怒），迦梨是“灭世的愤怒”（无限的愤怒）。两者都是雪山神女的化身，但杜尔迦强调“节制”，迦梨强调“毁灭”。

二、十臂与法器

2.1 十臂的象征

杜尔迦最显著的形象是十臂（象征无限能力），每只手持一件法器：

  • 三叉戟（象征湿婆的毁灭力）
  • 剑（象征智慧斩愚昧）
  • 弓箭（象征专注与目标）
  • 金刚杵（象征不可摧毁）
  • 法轮（象征时间之轮）
  • 绳索（象征束缚恶者）
  • 钟（象征觉醒）
  • 水罐（象征生命之源）
  • 盾（象征保护）
  • 莲花（象征纯净）

2.2 骑狮与穿红

杜尔迦的坐骑是狮子（象征勇猛），常穿红色纱丽（象征愤怒与力量）。在艺术中，她常被描绘为骑狮，十臂挥舞法器，周围环绕着战场。

2.3 斩杀水牛魔王

杜尔迦斩杀魔王摩醯湿（水牛形）的传说，成为印度“正义战胜邪恶”的永恒叙事。这个神迹在杜尔迦节中被反复演绎。

三、杜尔迦节（Durga Puja）

3.1 九天九夜的祭祀

每年秋季（公历9-10月），举行杜尔迦节（Durga Puja）：

  • 迎神：迎请杜尔迦像，供奉九天
  • 祭祀：每日念诵《往世书》颂诗，献祭动物
  • 送神：将杜尔迦像浸入水中，象征“愤怒归水，秩序重现”

3.2 西孟加拉的盛典

西孟加拉邦（Kolkata）的杜尔迦节是印度最盛大的节日之一，吸引数百万信徒参与。临时祭坛（Pandal）遍布城乡，艺术雕像精美绝伦。

杜尔迦节不仅是宗教节日，更是妇女力量的彰显。在 patriarchar 社会中，杜尔迦展示了“女性愤怒即力量”的永恒真理。

4.2 红装与金饰

杜尔迦常穿红色纱丽（象征愤怒与力量），佩戴金饰（象征胜利）。在印度艺术中，她是色彩最艳丽、气势最磅礴的女神。

4.3 水牛与战场

杜尔迦的象征物是水牛（魔王摩醯湿）与战场（象征正义之战）。在艺术中，她常被描绘为在战场上斩杀水牛魔王。

潘：希腊神话中的牧神与森林之神

前言

上古之时，希腊山野之间，有神降焉。

神名潘，父为赫尔墨斯，母为德律俄珀，山林女神也。潘初生时，羊角歧趾，遍体绒毛，母见而大骇，呼曰：“怪哉！“遂遁入林，不敢复认。父笑而抱之，献于奥林匹斯之巅。诸神见其状，皆曰：“此小羊孩也，可爱哉！“遂留于神山，为众神之宠儿。

潘稍长，居阿卡迪亚之深林，与宁芙牧人为伴。其状也，上半为人，下半为羊，头有弯角，足有分蹄。每行林间，吹芦苇之箫，七音悠扬，羊群随行，野兽驻足而听。其笛声者，林之声也；其人者，野之魂也。

有牧人问之曰：“汝名为何？“潘答曰：“吾乃潘，万物之灵也。“牧人以其形，半人半羊，乃称之为"潘格洛斯”，意云"全羊”。其名遂传于世。

潘有异能，能于酣睡中骤醒，长啸一声，其声凄厉，穿林越谷，闻者股栗，陷入莫名之惧。公元前四百九十年，波斯王率军来攻雅典，城危在旦夕。忽夜闻潘之啸声，波斯军大恐，以为其神兵天降，遂溃散而逃。雅典人转危为安，自此奉潘为护国之神。

潘创排箫之器，取七芦为之，音韵古朴，为田园之音。相传潘曾慕山林女神绪任克斯之美，追逐不及，女化为芦苇，潘乃取芦制箫，使女之声永留人间。

后基督之光普照大地，有传言曰：“潘死矣。“当其时，异教衰落，神话渐远，而潘之笛声，永留天地之间。

阿波：文莱巨人

老梵 发布于 2026-05-07 收录于 巨人专题

前言

在文莱神话里，阿波是婆罗洲的守护者，身体像基纳巴卢山一样大，曾与英国殖民者斗法，最后被《古兰经》感化。本文将用故事性的笔触，带你走进这位文莱巨人的传奇：从他镇守婆罗洲，到与殖民者斗法，再看他如何皈依伊斯兰教，成为守护神。

阿波：文莱巨人#

关键词：阿波、文莱神话、巨人专题、婆罗洲守护

一、镇守婆罗洲：山一样大的身体

阿波的身体像基纳巴卢山一样大，每一步踩下去，地面都裂开一道缝。他镇守着婆罗洲，不让外敌入侵。

“这是我的地盘！”阿波吼一声，声音像季风，“谁敢来，我就捏碎他！”

岛上的百姓都敬仰他，每年都来祭拜，求他保佑航运平安。

二、与殖民者斗法：魔法的较量

英国殖民者带着火炮来了，阿波喷出毒火，想把他们的船烧了。

“这怪物，太可怕了！”殖民者吓得后退，“用《古兰经》，只有真主的经文能破他的魔法！”

伊玛目念着《古兰经》，把经文洒向阿波。没想到，经文一碰到他的皮肤，他就安静了下来。

三、皈依伊斯兰教：成为守护神

伊玛目对阿波说：“你若皈依伊斯兰教，我就封你为婆罗洲的守护神。”

阿波叩首：“弟子愿皈依。”

从此，阿波成了婆罗洲的守护神。渔民们出海前，都会来祭拜他，说：“阿波保佑我们平安。”

四、巨人的 Legacy：基纳巴卢的传说

阿波死后，他的身体化作了基纳巴卢山。直到今天，文莱人还说：“那座山，就是阿波的化身。”

他的故事，成了文莱神话里“从反抗到皈依”的经典，也记录了伊斯兰教传入文莱的历史。

五、总结

阿波作为文莱神话里的巨人，用山一样的身体镇守婆罗洲，最终皈依伊斯兰教成为守护神。他的故事，既是文莱民族的守护传奇，也是“从反抗到融合”的宗教寓言，提醒我们：真正的强大，来自包容与信仰。

白素贞是江南神话中最著名的女性形象之一，源自《白蛇传》这一中国四大民间传说。她本是峨眉山修炼千年的白蛇精，为报许仙前世救命之恩，下山寻找许仙，与之结为夫妻。然而金山寺住持法海认为人妖殊途，设计拆散二人，最终将白素贞镇压于雷峰塔下。白素贞的故事跨越千年，从唐代传奇《白蛇记》到明代话本《白娘子永镇雷峰塔》，再到现代影视剧《新白娘子传奇》，不断演变丰富，成为江南文化乃至中华文化的标志性符号。本文将系统梳理白素贞的起源、修炼历程、与许仙的爱情、与法海的对抗、文化影响及现代传承，揭示这位千年白蛇如何成为爱情、执着与牺牲的永恒象征。
