阿努比斯：木乃伊与死者守护神

老梵 发布于 2026-05-06 收录于 非洲神话

前言

阿努比斯（Anubis）是古埃及神话中最具辨识度的神祇之一——豺狼头、黑皮肤、手持天平，他是木乃伊制作的保护神、死者心脏的称量者、冥界审判的守护者。作为奈芙蒂斯与奥西里斯（名义上是赛特）的儿子，阿努比斯连接了埃及神话的多个核心神祇，他的形象直接催生了埃及的木乃伊制度，也成为了人类对死亡最古老的仪式化表达之一。

阿努比斯：木乃伊与死者守护神

关键词：阿努比斯、木乃伊、死者守护神、埃及神话、奥西里斯、奈芙蒂斯、冥界审判、豺狼、天平

一、阿努比斯的起源与身世

1.1 奈芙蒂斯与奥西里斯之子

据《死者之书》记载，阿努比斯是天空女神努特之子，但父亲身份有争议。最广泛的说法是：奈芙蒂斯（赛特之妻）化装成伊西斯，引诱奥西里斯而生下阿努比斯。赛特发现后试图杀害他，但奈芙蒂斯将他交给伊西斯抚养。

1.2 名义上的父亲：赛特

虽然赛特不是亲生父亲，但作为奈芙蒂斯的丈夫，他名义上是阿努比斯的父亲。这种"名义父亲"的关系，使阿努比斯在赛特与荷鲁斯的战争中保持中立，成为公正的审判者。

1.3 早期的地位：第一冥王

在古王国时期（公元前2686-2181年），阿努比斯是主要的冥界之神。直到中王国时期，奥西里斯取代他成为冥界之王，阿努比斯降级为"冥界守护者"，但保留了木乃伊制作与心脏称量的职责。

二、木乃伊制作的保护神

2.1 第一个木乃伊：奥西里斯

神话记载，伊西斯复活奥西里斯时，阿努比斯协助用亚麻布包裹尸体，制作了人类历史上第一个木乃伊。从此，他成为木乃伊制作（Embalming）的保护神，祭司在制作木乃伊时戴阿努比斯面具。

2.2 墓地守护者

阿努比斯常被视为"墓地之主"（Khenty-imentiu），他的豺狼形象源于在墓地游荡的胡狼。埃及人认为，胡狼守护墓地不被盗掘，因此阿努比斯成为墓地的最高守护者。

2.3 开口仪式（Opening of the Mouth）

阿努比斯发明了"开口仪式"——用特殊工具触碰木乃伊的嘴，使其能呼吸、说话、进食于来世。这个仪式成为埃及葬礼的核心程序，持续了三千年。

三、冥界审判：心脏称量者

3.1 玛特之秤

在冥界杜亚特（Duat）的审判大厅，阿努比斯负责操作"玛特之秤"：将死者的心脏放在一端，另一端放玛特女神的真理羽毛。他是称量的直接执行者，奥西里斯则是最终裁决者。

3.2 心脏轻盈者：进入亚卢

如果心脏比羽毛轻（象征一生无罪），阿努比斯将死者引至奥西里斯面前，获准进入永恒乐土"亚卢"（Aaru）。这是埃及人来世信仰的核心。

3.3 心脏沉重者：被阿米特吞噬

如果心脏比羽毛重（象征罪孽深重），阿努比斯将死者交给怪物"阿米特"（Ammit，鳄鱼头、狮身上、河马下身）。阿米特吞食死者，使其灵魂"第二次死亡"（彻底消失）。

四、阿努比斯的象征与艺术表现

4.1 豺狼头与黑皮肤

阿努比斯最显著的特征是豺狼（胡狼）头、黑皮肤。黑色象征尼罗河淤泥（重生）和尸体防腐后的颜色。豺狼则源于在墓地游荡的胡狼，象征对死者的守护。

4.2 天平与权杖

阿努比斯常手持"玛特之秤"和"连枷"（Flail）。天平象征审判的公正，连枷象征对死者的保护（奥西里斯的圣器）。

4.3 金指甲与护身符

阿努比斯的金指甲用于触碰木乃伊的嘴（开口仪式）。埃及人制作阿努比斯护身符，相信能保护尸体不被盗墓者破坏，确保来世顺利。

五、阿努比斯崇拜中心

5.1 基诺坡利斯：阿努比斯之城

上埃及的基诺坡利斯（Cynopolis，意为"狗城"）是阿努比斯的主要崇拜中心。这里有巨大的阿努比斯神庙，祭司每年举行"阿努比斯节"，游行阿努比斯神像，祈求墓地安宁。

5.2 萨卡拉：木乃伊制作中心

萨卡拉（Saqqara）的地下墓穴是阿努比斯的重要圣地。这里埋葬着数百万只木乃伊胡狼，作为对阿努比斯的献祭。2023年，考古学家在此发现一座完整的阿努比斯神庙，内有纯金阿努比斯像。

5.3 阿拜多斯与底比斯

在阿拜多斯（奥西里斯圣地）和底比斯（卡纳克神庙），阿努比斯作为"冥界守护者"被共同崇拜。他的雕像常与奥西里斯、伊西斯并列，构成"神圣家庭"的延伸。

六、阿努比斯与其他神话体系

6.1 与希腊赫尔墨斯

希腊人将阿努比斯等同于赫尔墨斯（Hermes），但赫尔墨斯是信使之神，阿努比斯是死者守护神。后来两者融合为"赫尔曼努比斯"（Hermanubis），成为希腊-埃及的神秘信仰核心。

6.2 与罗马阿努比斯崇拜

罗马帝国时期，阿努比斯崇拜传入罗马。庞贝古城的阿努比斯神庙保存完好，维苏威火山爆发时，信徒还在庙中举行"开口仪式"。

6.3 对基督教末日审判的影响

阿努比斯的"心脏称量"审判，可能影响了基督教的"末日审判"观念。两者的结构高度相似：死后审判、称重/审查、天堂/地狱的二元结果。

七、阿努比斯在《死者之书》中的地位

7.1 第30章：阿努比斯咒语

《死者之书》第30章是"阿努比斯咒语"，死者必须背诵：“阿努比斯啊，请不要让我的心脏在天平上作伪证。“这确保了审判的通过。

荷鲁斯：天空与王权之神

老梵 发布于 2026-05-06 收录于 非洲神话

前言

荷鲁斯（Horus）是古埃及神话中最重要的鹰神，被视为法老的守护神与王权的象征。作为奥西里斯与伊西斯的儿子，他花了80年与叔叔赛特争夺王位，最终获胜成为人间之王。他的"荷鲁斯之眼"（Wedjat）成为埃及最强大的护身符，他的形象"鹰首人身"成为法老权威的终极象征，他的胜利开创了"法老即荷鲁斯化身"的埃及王权传统。

荷鲁斯：天空与王权之神

关键词：荷鲁斯、天空之神、王权之神、埃及神话、奥西里斯、伊西斯、赛特、法老、荷鲁斯之眼、

一、荷鲁斯的诞生与童年

1.1 冥界之种

奥西里斯被赛特杀害后，伊西斯找回13块尸块制成木乃伊，用魔法使其短暂复活，从而受孕生下荷鲁斯。因此，荷鲁斯既是冥界之王的后代，又是魔法复活的产物。

1.2 沼泽中的秘密成长

伊西斯带着婴儿荷鲁斯逃到尼罗河三角洲的沼泽中，躲避赛特的追杀。在这里，哈托尔（Hathor）用牛奶哺育他，奈芙蒂斯（Nephthys）保护他，图特（Thoth）教导他智慧。

1.3 成年与复仇誓言

荷鲁斯成年后，在九神团面前宣誓夺回王位。他的誓言标志着埃及王权从冥界（奥西里斯）向人间（荷鲁斯）的转移，也开启了"法老即荷鲁斯"的永恒传统。

二、与赛特的80年战争

2.1 王位争夺的开始

荷鲁斯向九神团提出挑战，要求从叔叔赛特手中夺回属于父亲的王位。赛特嘲笑他年轻，但九神团同意举行审判。

2.2 多种形态的搏斗

这场战争持续80年，包含多种神奇搏斗：

  • 河马搏斗：两神变作河马在尼罗河中厮杀
  • 植物比赛：赛特种下莴苣（象征同性恋），试图羞辱荷鲁斯
  • 明珠诱惑：赛特变作明珠，试图从内部炸毁荷鲁斯
  • 金属鳄鱼：荷鲁斯变作金属鳄鱼，咬碎赛特的武器

2.3 荷鲁斯之眼的失去与治愈

在一次战斗中，赛特挖去了荷鲁斯的左眼（荷鲁斯之眼）。图特用瞪羚的乳汁治愈了这只眼，使其成为"完整、健康、保护"的象征，广泛用于护身符。

三、最终审判与胜利

3.1 九神团的裁决

经过80年战争，九神团（包括拉、舒、泰芙努特等）最终裁定荷鲁斯获胜。赛特被贬为"沙漠之王"，但仍保留对抗阿佩普的职责。

3.2 上下埃及的统一

获胜后，荷鲁斯将上埃及（秃鹰女神涅赫贝特Nekhbet）与下埃及（眼镜蛇女神瓦杰特Wadjet）统一，这就是"双冠"（Pschent）的由来，象征全埃及的王权。

3.3 成为法老的守护神

一、赛特的起源与象征

1.1 努特与盖布之子

赛特是天空女神努特（Nut）与大地之神盖布（Geb）的儿子，奥西里斯、伊西斯、奈芙蒂斯的兄弟。他诞生时撕裂了母亲的腹部，象征他"打破秩序"的本性。

1.2 动物形象：赛特兽

赛特的形象是独特的"赛特兽"——一种无法归类的神兽，有弯曲的鼻子、分叉的尾巴、直立的耳朵。学者认为这可能是土豚、驴、貘的混合体，象征"不属于任何秩序"的混沌属性。

  • 风暴权杖（Was-scepter）：顶端有赛特兽头，象征他的权威
  • 沙漠陶器：赛特是沙漠之神，陶器象征荒芜与外来
  • 闪电与雷鸣：他通过风暴展现混沌力量

二、弑兄与夺位

2.1 嫉妒的源头

赛特嫉妒兄长奥西里斯的威望与贤明统治。据《死者之书》记载，赛特设计了一场宴会，制作了精美棺材，诱骗奥西里斯躺入后封死，投入尼罗河。

2.2 分尸十四块

赛特抓住奥西里斯的尸体，将其分尸为14块，抛散在埃及各地。他试图通过这种方式，让奥西里斯永远无法复活，确保自己的王位。

2.3 与伊西斯的对抗

伊西斯搜寻回13块尸块（生殖器被鱼吞食），复活了奥西里斯。从此，赛特与伊西斯成为死敌，象征着混沌与魔法、破坏与重生的永恒对抗。

三、帮助拉神对抗阿佩普

3.1 每夜的搏斗

尽管赛特是弑兄者，但他每晚在冥界帮助拉神对抗混沌蛇怪阿佩普（Apep）。他用长矛刺伤阿佩普，确保太阳船能通过，展现了"以混沌对抗混沌"的悖论。

3.3 赛特与风暴的净化

赛特通过风暴带来沙漠的沙尘，但也用风暴清洗尼罗河的淤泥。这种"破坏即净化"的双重性，成为埃及神学的重要命题。

九神团最终判决荷鲁斯获胜，成为人间法老的守护神。赛特被贬为"沙漠之王"，但保留对抗阿佩普的职责，体现了"恶有恶用"的埃及智慧。

当赛特杀害并分尸奥西里斯后，伊西斯用魔法搜寻回13块尸块，用亚麻布包裹，施展"开口仪式"（Opening of the Mouth），使奥西里斯短暂复活，并因此受孕生下荷鲁斯。这是木乃伊技术的神圣起源。

2.3 保护荷鲁斯

在荷鲁斯与赛特争夺王位的80年战争中，伊西斯多次用魔法保护儿子。她召唤图特（Thoth）治疗荷鲁斯被挖去的眼睛，创造了"荷鲁斯之眼"（Wedjat）——完整与保护的象征。

3.2 生育与哺乳的守护

埃及妇女在分娩时呼唤伊西斯，相信她会消除难产之痛。伊西斯像常表现为抱婴孩荷鲁斯的母亲形象，这种"圣母子"构图后来被早期基督教挪用为圣母玛利亚与耶稣。

3.3 生命之符（Ankh）

伊西斯常手持生命之符（Ankh），象征她赋予生命的能力。在 temple of Isis at Philae 的浮雕中，她将生命之符送到法老唇边，赋予其永生。

四、王权的守护者

4.1 法老的加冕仪式

每个法老加冕时，都要举行"伊西斯拥抱"仪式：祭司扮作伊西斯，将王冠戴在法老头上，象征"王权来自伊西斯"。这确立了法老统治的神圣合法性。

4.2 生命之屋（House of Life）

伊西斯建立了"生命之屋"——埃及的魔法与医学中心。祭司们在这里抄写魔法纸草（Papyri），学习治疗、占卜、护身符制作。这成为后来亚历山大图书馆的前身。

4.3 与拉神的融合

新王国时期，伊西斯与拉神融合为"伊西斯-拉特"（Isis-Raet），成为太阳神之母。她不仅守护人间法老，更守护天上的太阳神，地位达到顶峰。

五、伊西斯崇拜的扩张

5.1 从埃及到希腊

托勒密王朝时期，伊西斯崇拜传入希腊。希腊人将其等同于得墨忒耳（Demeter），建立了融合式的伊西斯-得墨忒耳神庙。亚历山大港成为伊西斯信仰的新中心。

5.2 罗马帝国的国教

罗马帝国时期，伊西斯崇拜达到顶峰。庞贝古城的伊西斯神庙保存完好，维苏威火山爆发时，信徒还在庙中祈祷。罗马皇帝卡利古拉、提比略都曾是伊西斯信徒。

5.3 神秘宗教的巅峰

伊西斯信仰是古代世界最成熟的神秘宗教之一。入教仪式包括黑暗试炼、洗礼、圣餐，承诺信徒"获得永生"。这与后来的基督教仪式有惊人的相似性。

六、艺术表现与象征

6.1 王座头饰

伊西斯最显著的特征是头戴王座（她的名字象形文字就是王座）。这象征她是"王权的源泉"，法老坐在她赐予的王座上统治。
