嫦娥：月宫仙子

老梵 发布于 2026-05-13 收录于 中国神话

前言

后羿射下九个太阳，成为万民敬仰的英雄之后，他的妻子嫦娥从此走进了世人的视野。嫦娥，这个名字在四千年后依然被无数人传唱，可那些传说背后的故事，却比史书上的寥寥数语更为沉重。她本是凡间一个寻常女子，因缘际会嫁给了那位神射手，度过了人生中最幸福的时光。然而平静的日子没有持续太久——后羿求仙问道三年，从西王母处求得不死药，想要与嫦娥永远在一起。两颗药丸，一大一小，他本想先后飞升，同登仙籍。可蓬蒙的出现打破了一切，那个贪婪的徒弟闯入她的卧室，持剑逼她交出药物。那一夜，她做出了一个决定——将两颗药丸尽数吞下。她的身体越来越轻，越升越高，从地面升到云端，从云端飞向那轮皎洁的明月。从此，她成了月宫仙子，成了广寒宫的主人，在那永恒的孤寂中日复一日地悔恨、日复一日地思念。日遇七十毒的神农尝百草是为了族人，而嫦娥吞下灵药，却是因为爱与绝望。她与后羿天各一方，再也无法回到人间。碧海青天，夜夜心寒。这就是她的故事。

伐楼拿：水神与宇宙秩序

老梵 发布于 2026-05-07 收录于 印度神话

前言

在印度吠陀神话体系中，伐楼拿（Varuna）是最具秩序色彩的水神，据《梨俱吠陀》记载，他“坐于水中，掌管宇宙秩序（Rita）”，监督人类行为，惩罚违背秩序者。作为吠陀三大主神之一（因陀罗、阿耆尼、伐楼拿），他深深影响了印度的水崇拜、法律观念、道德约束与宇宙哲学，成为“秩序即宇宙”的永恒象征。

伐楼拿：水神与宇宙秩序

关键词：伐楼拿、水神、宇宙秩序、吠陀、印度神话、Rita、因陀罗、阿耆尼

一、伐楼拿的起源与地位

1.1 吠陀三大主神之一

据《梨俱吠陀》记载，伐楼拿是吠陀三大主神之一：

  • 因陀罗：雷神、战神、主神
  • 阿耆尼：火神、祭司之神
  • 伐楼拿：水神、宇宙秩序维护者

伐楼拿在吠陀早期的地位甚至高于因陀罗，后来逐渐被毗湿奴超越。

1.2 宇宙秩序（Rita）的维护者

伐楼拿最重要的职责是维护“Rita”（宇宙秩序、自然法则）：

  • 天律：太阳东升西落、四季更替
  • 地律：河水流动、植物生长
  • 人律：道德行为、祭祀规范

1.3 与因陀罗、阿耆尼的对比

因陀罗代表“雷霆征服”，阿耆尼代表“火祭转化”，伐楼拿代表“水与秩序”。三者构成吠陀“雷-火-水”的宇宙三角。

二、Rita：宇宙秩序的核心

2.1 Rita的含义

Rita（梵语：Ṛta）是“宇宙秩序、自然法则、道德律令”的总称：

  • 物理秩序：天体运行、季节更替、水流方向
  • 道德秩序：诚实、守誓、祭祀规范
  • 社会秩序：种姓职责、君王正法

2.2 伐楼拿的监督

伐楼拿“千眼”（Sahasraksa）监视人间一切行为：

  • 记善：遵守Rita者，赐福健康、财富
  • 记恶：违背Rita者，降下疾病、旱灾
  • 忏悔：诚心忏悔者，伐楼拿可宽恕

2.3 对后世法律的影响

伐楼拿维护Rita的观念，深深影响了印度的法律传统（Dharma-shastras）。《摩奴法典》（Manusmriti）就是Rita观念的法律化。

三、伐楼拿的象征与艺术表现

3.1 千眼与水罐

伐楼拿最显著的象征是“千眼”（象征全视）与水罐（象征水源）。在吠陀艺术中，他常被描绘为坐于水中，周围环绕着千眼。

3.2 绳索与束缚

伐楼拿手持绳索（Pasha），用来捆绑违背Rita者。在艺术中，他常被描绘为用绳索束缚罪人，象征“秩序对混乱的约束”。

3.3 坐骑与伴侣

伐楼拿的坐骑是海兽（Makara，鳄鱼与大象的混合体），象征深海力量。他的伴侣是伐楼尼（Varuni，酒神），象征水与醉意的结合。

四、伐楼拿的崇拜中心

4.1 吠陀祭祀（各地）

伐楼拿是吠陀祭祀（Yajna）的重要对象。祭司念诵《梨俱吠陀》中伐楼拿的颂诗（共40多首），祈求水源充足、秩序井然。

4.2 恒河崇拜

恒河（Ganga）被视为伐楼拿的居所。每年举行“恒河灯节”（Ganga Dussehra），信徒沐浴恒河，祈求伐楼拿宽恕罪业。

4.3 现代吠陀传承

在尼泊尔、斯里兰卡等吠陀传承地区，伐楼拿仍是重要的水神与秩序神。每年举行“伐楼拿祭”（Varuna Jatra），祈求风调雨顺。

五、与其他水神的对比

5.1 与希腊波塞顿

希腊波塞顿是海神（三叉戟），伐楼拿是水神与秩序神。两者都掌管水域，但波塞顿强调“狂暴”，伐楼拿强调“秩序与监督”。

5.2 与中国水神共工

中国水神共工是洪水之神（怒触不周山），伐楼拿是秩序之神。两者都涉及水，但共工代表“混乱”，伐楼拿代表“秩序”。

5.3 与埃及努恩

埃及努恩（Nu）是原始水（混沌），伐楼拿是秩序水神。两者都代表水，但努恩是“混沌之水”，伐楼拿是“秩序之水”。

六、对印度文明的影响

6.1 法律传统的奠基

伐楼拿维护Rita的观念，成为印度法律传统（Dharma-shastras）的基石。《摩奴法典》就是Rita观念的法律化。

安丹：非洲班图巨人

老梵 发布于 2026-05-07 收录于 巨人专题

前言

在非洲班图神话里，安丹是草原上的巨人，脚掌像芭蕉叶一样大，能举起整座山脉当武器。本文将用故事性的笔触，带你走进这位非洲巨人的传奇：从他在大草原上的行走，到众神用蜂蜜酒灌醉他，再看这位巨人的末日，理解非洲人对自然力量的敬畏。

安丹：非洲班图巨人#

关键词：安丹、非洲神话、班图神话、巨人专题、草原巨人

一、草原上的巨足：每一步都震裂大地

安丹的脚掌像芭蕉叶一样大，每一步踩下去，草原上就多了一个深坑。他弯腰捡起一座山脉，当棍子挥舞，把天上的云都扫散了。

“这草原太小了，”安丹嘟囔着，“得找点更大的地方。”

他的汗水流下来，变成了赞比西河；他的呼吸吹过，草原上的草木都弯了腰；他的影子投在地上，遮住了整个村庄。

二、众神的恐惧：草原的威胁

众神在天上看着安丹，心里都发毛：“这巨人要是把整个非洲都踩碎了，我们住哪里？”

天神穆伦圭对众神说：“得想办法杀了他，不然草原永无宁日。”

可安丹太强了，徒手能撕开狮子，一口能吞下一头大象。众神商量了三天三夜，终于想出一条计策：“用蜂蜜酒灌醉他！”

三、蜂蜜酒的陷阱：巨人的贪杯

众神派了最美的女神涅尼，带着十桶蜂蜜酒去找安丹。

“伟大的巨人，”涅尼笑着说，“这是天神特酿的蜂蜜酒，请你品尝。”

安丹眼睛都亮了：“好香的酒！”他一口干了第一桶，又干了第二桶，一口气喝了八桶，终于醉得东倒西歪。

“再来……再来……”安丹嘟囔着，轰然倒地，鼾声震得草原上的鸟儿都飞了。

四、巨人的末日：众神的围杀

众神看到安丹醉倒，纷纷拿出武器冲了下来。

穆伦圭用雷锤砸他的脑袋，铁匠神用烧红的铁钎刺他的眼睛，草原神用藤蔓绑住他的手脚。

安丹在梦里挣扎：“谁……谁在打我……”可蜂蜜酒太烈了，他连眼睛都睁不开，就被众神活活打死了。

他的尸体倒在地上，变成了非洲的第二个高原；他的血液变成了红土，滋养了这片大地。

五、总结

安丹作为非洲班图神话里的草原巨人，用巨大的脚掌震裂大地，最终被众神用蜂蜜酒灌醉后杀死。他的故事，既是非洲人对自然力量的敬畏，也是“再强大的巨人，也敌不过智慧与阴谋”的寓言，提醒我们：力量再大，也抵不过策略与协作。

足名椎与手名椎：栉名田比卖的父母

关键词：足名椎命、手名椎命、栉名田比卖、八岐大蛇、须佐之男、出云、土地神、父母神

1.1 身世与神格

作为地方性的土地神，足名椎命与手名椎命没有天照大神（天照大神，あまてらすおおみかみ）那般高高在上，也没有须佐之男那般狂暴不羁。他们更贴近当地居民的生活，负责保佑风调雨顺、五谷丰登、家宅平安。

1.2 家庭与后代

足名椎命与手名椎命一共生了八个女儿，这在古代日本神话中象征着多子多福、家族兴旺。然而，这份幸福却被当地的一条凶恶大蛇打破了。八岐大蛇每年都会要求足名椎命一家献上一个女儿作为祭品，否则就摧毁整个地区。

八岐大蛇（八岐大蛇，やまたのおろち）是出云地区最恐怖的妖怪，它有八个头、八条尾，身长足以跨越八座山峰，背上生长着青苔、杉树与柏树。它的眼睛像红灯笼般赤红，每年都会从居住的斐伊川（斐伊川，ひいかわ）上游游下来，要求当地居民献上少女作为祭品。

2.2 七个女儿的悲剧

面对八岐大蛇的威胁，足名椎命与手名椎命不得不忍痛将女儿们逐年献上。大女儿、二女儿……直到七女儿，全部被八岐大蛇一口吞食。每当女儿被带走，足名椎命与手名椎命都会痛哭流涕，但面对强大的妖怪，他们毫无反抗之力。

到了第八年，足名椎命与手名椎命只剩下最小的女儿栉名田比卖（櫛名田比売，くしなだひめ）。她是八姐妹中最美丽的一个，也是父母最后的希望与慰藉。然而，八岐大蛇又来了，要求将栉名田比卖也献上。足名椎命与手名椎命悲痛欲绝，知道这可能是最后一次见到小女儿。

就在栉名田比卖即将被献给八岐大蛇的那一年，被姐姐天照大神放逐的须佐之男命（須佐之男命，すさのおのみこと）漫游来到了出云国。他因在高天原的暴行而被放逐，却正好在足名椎命一家最绝望的时刻降临。

须佐之男听到足名椎命与手名椎命的痛哭声，上前询问缘由。得知八岐大蛇的暴行与将要失去最后一个女儿的悲剧后，须佐之男说道：“如果你们将女儿嫁给我，我就斩杀这条大蛇。”

3.3 父母的承诺

足名椎命与手名椎命听到这番话，如同抓住了救命稻草。他们立刻同意将栉名田比卖许配给须佐之男，只求他能斩杀八岐大蛇，结束这长达七年的噩梦。

4.2 斩杀大蛇

须佐之男见状，拔出腰间十拳剑（十拳剣，とっきのみつるぎ），将八岐大蛇斩成碎段。在斩断大蛇的尾巴时，剑锋突然崩缺，须佐之男好奇地剖开蛇尾，发现其中藏有一把更为锋利的神剑——天丛云剑（天叢雲剣，あめのむらくものつるぎ）。

4.3 成功与嫁女

八岐大蛇被斩杀后，足名椎命与手名椎命终于从长达七年的恐惧中解脱出来。他们按照承诺，将栉名田比卖嫁给须佐之男。这场婚姻不仅拯救了他们的女儿，更开启了出云系谱的延续，最终诞生了大国主神（大国主神，おおくにぬしのかみ）。

五、须贺宫的建立

5.1 新婚宫殿

为了纪念这段姻缘，须佐之男在出云须贺建造了壮丽的宫殿，命名为"须贺宫"（須賀宮，すがのみや）。宫殿建成后，栉名田比卖正式成为须佐之男的妻子，足名椎命与手名椎命也成为这位征服者的岳父母，在出云地区地位大增。

5.2 作为岳父母的角色

作为须佐之男的岳父母，足名椎命与手名椎命在须贺宫中扮演着重要角色。他们负责教导栉名田比卖如何管理宫廷事务，接待各方神灵，成为出云地区"第一家庭"的长辈。

6.1 外祖父母

足名椎命与手名椎命是栉名田比卖的父母，因此也是须佐之男的岳父母。栉名田比卖生下八岛士奴美神（八島士奴美神，やしまじぬみのかみ），经过数代传承，最终诞生了大国主神。因此，足名椎命与手名椎命是大山津见神（大山津見神，おおやまつみのかみ）的亲家，也是大国主神的六世外祖父母。

6.2 默默无闻的守护者

七、文化影响与后世传承

7.1 须贺地区的信仰

在岛根县松江市须贺町一带，当地居民至今仍供奉着足名椎命与手名椎命。他们将这两位神祇视为保护家园、驱除灾害的守护神。每年的祭祀活动中，都会讲述他们如何用坚持与智慧，为女儿换来幸福的故事。

7.2 文学与艺术

足名椎命与手名椎命的故事是日本古典文学中"父母之爱"的典范：

  • 《古事记》与《日本书纪》详细记载了他们失去七个女儿的悲痛
  • 能剧《八岐大蛇》中，他们是被同情的对象，也是须佐之男拯救行动的核心
  • 浮世绘中经常出现他们痛哭流涕，恳求须佐之男拯救女儿的场景

7.3 现代意义

在现代日本，足名椎命与手名椎命的故事常被用来比喻"父母对子女的无限爱护"。他们面对强权（八岐大蛇）的无助，以及对拯救者（须佐之男）的感激，深深触动了每一个为人父母的心。

八、与其他神话父母的比较

8.1 与伊邪那岐、伊邪那美的比较

伊邪那岐（伊邪那岐命，いざなぎのみこと）与伊邪那美（伊邪那美命，いざなみのみこと）是创世神，共同创造了日本列岛；足名椎命与手名椎命是地方土地神，只守护出云须贺的一小片土地。前者是神族的祖先，后者是凡人的守护者。

前言

在那混沌初开之时，泰坦之主克洛诺斯以吞噬子女闻名。波塞冬便是这被吞入父腹的第二子，与兄弟宙斯、哈迪斯一同在黑暗中等待重见天日的命运。

待克洛诺斯被自己的子女推翻，三兄弟抽签分天下：宙斯得天空，哈迪斯得冥界，而波塞冬——他抽中了深不可测的大海。自此，他成为海洋的主宰，执掌万水之域，以三叉戟为器，能唤起滔天巨浪，亦能撼动大地令地震频发。

他的威严不仅存于海上。曾有一日，他化作一匹神骏的白马，与美杜莎相遇，由此诞下了飞马喀罗陀斯与巨人喀盛诺耳——那是海洋之力与死亡之眼交织而生的后裔。

奥林匹斯山众神之中，波塞冬以易怒著称。他的愤怒能掀起海啸，令舰队覆没；他的叹息化作暗涌，使渔民迷失归途。希腊人向他献祭，并非出于敬爱，而是出于敬畏——他们知道，这位海神的脾性如同大洋，深不见底，变幻莫测。

天空有雷电为宙斯增辉，冥界有亡灵供哈迪斯统御，唯有大海永远沉默，在阳光无法穿透的深处，波塞冬独自统治着他的深渊。

这是他的故事。

燧人氏：中国神话中的取火之神

关键词：燧人氏、取火之神、中国神话、钻木取火、熟食时代、三皇

前言

天地初开，混沌之后，人类尚不知火为何物。他们如野兽般群居，生吞血肉，饮冷水，在寒夜中瑟缩发抖，在黑暗中被猛兽追逐。那是一个没有光芒、没有温暖、没有熟食的时代。

华胥氏之后，有一人降生于世，名曰燧人。他生而有神异，不甘于现状，矢志为人类找到光明与温暖。某日，森林大火过后，他尝到了被火烤熟的兽肉，那股焦香与温热令他久久不能忘怀。他走遍山川河海，终在一片燧木林中驻足——他看见鸟雀以喙啄木，迸出火星，由此悟得摩擦生热之理。

自此，燧人以燧木为钻，以软木为杆，日复一日地摸索尝试。手掌磨出血泡，他不曾放弃；无数次失败，他不曾气馁。终于，在一个寂静的夜里，他钻出了第一簇火苗。那微弱的火光，照亮了人类文明的曙光。

火从此留在人间。人类告别了茹毛饮血的蒙昧时代，进入了熟食文明的新纪元。后世尊他为取火之神、三皇之一，与伏羲、神农并称，世代祭祀，香火不绝。

这是他的故事。
