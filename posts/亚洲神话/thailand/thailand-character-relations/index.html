前言

泰国神话以《拉玛坚》（Ramakien）为核心，融合本土信仰、上座部佛教、华人移民神话，构成了一个“王权-神猴-蛇鸟-佛教”的人物关系网络。从拉玛（Phra Ram）作为曼谷王朝的神化象征，到神猴哈努曼（Hanuman）的忠诚与力量，再到本土蛇神纳迦（Naga）与鸟神迦楼罗（Garuda）的对立，以及佛陀与观音的融合，泰国神话展现了一个多元文化交织的独特世界。本文将系统梳理泰国神话核心人物关系，揭示其“王权神化-英雄传奇-多元融合”的三重架构。

泰国神话人物关系考：从拉玛到哈努曼的谱系网络

一、《拉玛坚》核心人物

泰国版《罗摩衍那》称为《拉玛坚》，核心人物已完全本土化：

拉玛（Phra Ram） 泰国版罗摩，被视为曼谷王朝拉玛一世至十世的名号来源，象征王权与正义。在《拉玛坚》中，拉玛是阿逾陀国王子，被流放森林，在哈努曼帮助下救回妻子悉达。

悉达（Nang Sida） 泰国版悉多，拉玛之妻，象征忠贞与纯洁。在泰国古典舞剧“孔剧”（Khon）中，悉达的面具为绿色，代表纯洁。

哈努曼（Hanuman） 神猴，拉玛的忠实盟友，拥有飞天、变身等神通。在泰国，哈努曼被视为保护神与力量象征，常出现在寺庙门神画像中。

二、本土神灵：纳迦与迦楼罗

纳迦（Naga） 蛇神，掌管水域与保护，泰国寺庙常有多头纳迦雕像，清迈帕辛寺有巨型纳迦守护寺门。

迦楼罗（Garuda） 鸟神，毗湿奴的坐骑，在泰国是王权的象征，国王的御用图案常为迦楼罗。

三、佛教与华人信仰的融合

佛陀（Phra Phuttha） 上座部佛教在泰国为国教，佛陀被视为最高精神导师，其形象遍布寺庙与家庭佛龛。

观音（Kuan Eim） 华人移民带来，在泰国被广泛崇拜，成为慈悲的象征，常与本土神灵融合供奉。

哪吒（Na Ja） 华人移民带来，在泰国被视为儿童保护神。

四、其他重要人物

因陀罗（Phra In） 雷电之神，在泰国被视为守护神，常在寺庙壁画中出现。

毗湿奴（Phra Narai） 维持之神，在泰国被视为保护神，其形象常与拉玛结合。

五、神话人物关系网络总结

泰国神话人物关系可归纳为四大网络：

  1. 王权神化网络：拉玛 → 曼谷王朝名号 → 王权与正义象征
  2. 英雄传奇网络：拉玛 + 哈努曼 → 救悉达 → 正义战胜邪恶
  3. 本土神灵网络：纳迦 ↔ 迦楼罗 → 蛇与鸟的对立
  4. 多元融合网络：佛陀 + 观音 + 哪吒 → 佛教与华人信仰融合
