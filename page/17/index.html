王母娘娘：瑶池金母与长生女神

老梵 发布于 2026-05-07 收录于 中国神话

前言

王母娘娘是民间信仰中最受爱戴的长生女神，她又被尊称为瑶池金母，是天庭之中最慈祥的母亲。她掌管着昆仑山上的蟠桃园，那园中的仙桃三千年一熟、六千年一熟、九千年一熟，吃了可与天地同寿。每年农历三月初三，她会举办盛大的蟠桃大会，邀请三界众仙赴宴品尝仙桃。她是玉皇大帝的贤内助，两人共同治理天庭，一阴一阳，相辅相成。她有七位美丽的女儿，人称七仙女，个个聪慧善良。在民间百姓心中，王母娘娘不仅是一位高高在上的女仙，更是一位有求必应的慈祥母亲——人们向她祈求长寿安康、子嗣绵延、家庭和睦。她的故事跨越了数千年，从《山海经》中的西王母演变而来，最终成为中国民间信仰中最温暖、最令人亲近的女神形象。

关键词：王母娘娘、瑶池金母、蟠桃园、中国神话、长生不老、七仙女、蟠桃会、玉皇大帝、民间信仰

一、我的起源

1.1 我最初的模样

说起我的来历，孩子们总是好奇得不得了——“王母娘娘，您是怎么成为神仙的呀？“每次听到这样的问题，我都会忍不住笑起来。

其实在很久很久以前，我并不叫王母娘娘。那时候的人们叫我西王母，住在遥远的昆仑山上。你们可知道最初的我在人们眼里是什么样子吗？《山海经》里说，我"其状如人，豹尾虎齿而善啸”——就是说我长着豹子的尾巴、老虎的牙齿，还特别擅长大声吼叫。每次读到这段描述，我自己都觉得有趣得很。

但你们不要害怕。那个凶猛的西王母，是远古时期人们对大自然力量的敬畏与想象。在那个年代，山川有灵，万物有神，人们需要一位掌管瘟疫与刑罚的威严神灵来维护天地的秩序。我只是穿上了那副威严的外衣，替天行道罢了。

随着岁月的流逝，人们渐渐发现，我不只是威严的。当他们虔诚地向我祈祷时，我能听见他们的声音，能感受到他们的苦难与渴望。于是，我开始慢慢变化——由凶神变成了慈神，从掌管死亡到赐予长生。这就是为什么后来的画像中，我头戴凤冠、身穿霞帔，手捧蟠桃，满脸慈祥。

1.2 瑶池边的修行

昆仑山是天地之间的神山，而瑶池就在昆仑之巅。那池水碧蓝如镜，甘甜如蜜，终年云雾缭绕，四季花开不败。我就是在瑶池边度过了一段漫长的岁月，修成了无上的仙道。

在瑶池边修行的日子里，我每天做的事很简单——照顾瑶池边的花草，聆听天地之间的声音，感受宇宙运行的规律。我渐渐明白，真正的长生不是活多久的问题，而是心性有多宽广的问题。一个只关心自己的人，就算活一万年也是短暂的；一个能与天地万物同心的人，哪怕只有一瞬也是永恒的。

瑶池的每一朵花、每一株草，都承载着我的修行。我在它们身上看到了生命的循环——花开会谢，叶落会生，看似有生有灭，实则生生不息。我就是在这样的领悟中，从西王母慢慢变成了你们所熟悉的王母娘娘。

1.3 嫁与玉皇大帝

后来，我的故事与天界另一位至尊的存在连接在了一起——那就是玉皇大帝。

你们一定很好奇，我是怎么成为玉皇大帝的妻子的。其实在道教的最高神理中，我与玉帝并非世俗意义上的夫妻，而是阴阳二气所化——他代表阳刚与秩序，我代表阴柔与长生；他掌管三界的运转，我掌管仙籍与长生。但在民间百姓的心中，天庭也应该像人间一样，有一位父亲、一位母亲，这样才完整，才温暖。

于是人们把我与玉帝说成了一家人——他是天公，我是地母；他是天庭的皇帝，我是瑶池的金母。我们两人共同治理天庭，就像人间的皇帝与皇后一样。这是百姓们的美好心愿，我怎么会忍心辜负呢？

其实，我与玉帝的关系远比夫妻更加深厚——我们是天地的两极、阴阳的平衡。没有他的阳刚，天庭就没有秩序；没有我的阴柔，天庭就没有温暖。我们彼此相补，共同维系着三界的和谐。

1.4 从西王母到王母娘娘

很多孩子会问我：“您到底是西王母，还是王母娘娘？“这个问题看似简单，却包含了一段漫长的演变过程。

最初的西王母，是《山海经》中那位住在昆仑山上的凶神，掌管瘟疫与刑罚。到了汉代，人们开始将我视为赐予不死药的女神——后羿从昆仑山求得不死药，嫦娥吃了飞上月宫，那不死药就是我的。到了魏晋南北朝时期，道教把我纳入了神系，尊我为"金母”，与东王公相对，共同管理神仙的品级与升迁。

真正转变为"王母娘娘”，是宋元以后的事。随着民间信仰的兴盛和戏曲小说的传播，我渐渐从一位神秘的昆仑女神，变成了天庭中那位慈祥的"王母娘娘”——玉皇大帝的妻子、七仙女的母亲、蟠桃会的主人。这是老百姓赋予我的全新身份，比起高高在上的西王母，我更喜欢"王母娘娘"这个称呼，因为它让我感觉自己离你们更近。

二、我的蟠桃园

2.1 园中的仙桃树

如果要问我天庭中最喜欢的地方是哪里，我一定会说——蟠桃园。

那片园子在昆仑山的东麓，占地不知多少亩，一眼望去全是层层叠叠的桃树。春天一到，满园的桃花盛开，粉红一片，香飘万里，那景象美得让人心醉。每一棵桃树都比我年纪还大——我说的是我成为神仙之后的年纪，不是那个"豹尾虎齿"的年纪。

园中一共有三千六百株桃树，分三种。前面一千二百株，花微果小，三千年一熟，人吃了能得道成仙、身体轻盈。中间一千二百株，层花甘实，六千年一熟，人吃了能长生不老、脱胎换骨。后面一千二百株，紫纹缃核，九千年一熟，人吃了能与天地同寿、与日月同庚。

每次我走在蟠桃园里，看着那些沉甸甸的桃子挂在枝头，心里就涌起一阵莫名的感动。这些桃树啊，见证了天庭的兴衰，见证了多少神仙的来来去去。它们不说话，却什么都知道。有时候我会坐在桃树下，轻轻摸摸树干，它们就像我的老朋友一样，默默陪着我。

2.2 种桃的辛苦

很多人以为蟠桃天生就是那么甜、那么大、那么神奇，其实不是的。种蟠桃比种人间的桃子辛苦一万倍。

每三千年开花结果一次，这中间要经历多少风吹日晒、霜打雨淋？桃树虽然长在仙界，但也需要精心照料。我要请专门的土地神守着园子，要请天界的甘露日日浇灌，要请太阳星君适时照耀，要请月宫中的嫦娥撒下清辉。这些工序，少一样都不行。

更不用说那些调皮的小仙童了。他们总是趁我不注意偷偷溜进园子里，想摘个桃子解解馋。每次被我抓到，他们就低着头，嘟着嘴，可怜巴巴地看着我。我哪里狠得下心来责罚他们？只能板着脸说：“这是给蟠桃会准备的，你们要是有道行，等蟠桃会上自然有你们一份。“他们这才欢天喜地地跑了。

还有孙悟空的事——我知道你们都想问这个。那猴子当年大闹蟠桃园，把满园的蟠桃糟蹋了不少。说实话，我确实生气了一阵子。但后来想想，那猴子天性跳脱，并非坏心，只是不懂规矩罢了。再说了，要不是他闹那一场，你们也不会有这么多关于我的传说流传下来，不是吗？

2.3 蟠桃的意义

在我的心里，蟠桃从来不只是桃子。它代表了长生，更代表了希望。

你想想看，一个人活在世上，最怕的是什么？是死亡，是离别，是看不到明天。而蟠桃给了人们一个希望——只要心存善念，勤加修行，就有机会超脱生死，获得永恒。这不是我自私地想要人们来膜拜我，而是我真的希望每一个善良的灵魂都能得到安宁与长久。

人间的桃子，一年一熟；天庭的蟠桃，三千年到九千年一熟。这其中的差距，其实就是凡与仙的距离。但距离并不代表无法跨越——只要心中有道，脚下有路，人人都可以向着仙境迈进。这就是蟠桃园存在的意义，也是我作为长生女神的使命。

三、蟠桃大会

3.1 筹备的忙碌

每年农历三月初三，是我的生辰，也是蟠桃大会的日子。这一天，三界十方、四生六道的神仙们都会赶来赴宴，场面热闹非凡。

为了这一天，我要提前半年就开始准备。蟠桃园里的桃子要一一检查，挑出最大最甜的那些。天厨的御宴要精心安排，每一道菜都要色香味俱全。瑶池里的甘泉要提前打上来醒着。七彩祥云要吹散得恰到好处，让整个天庭都沐浴在祥瑞的光芒中。

我的七仙女们也会来帮忙。大女儿负责安排座位，二女儿负责布置宴席，三女儿负责招待来宾，四女儿负责登记礼物，五女儿负责管束那些不听话的小仙童，六女儿负责给老仙人们添酒，小女儿——就是织女——负责歌舞表演。

你们可别小看这些准备工作。天庭不比凡间，来赴宴的神仙有三山五岳的山神土地，有四海龙宫的龙王太子，有各路星君、各方天尊。有些人性格冷傲，有些人热情好客，有些人喜欢安静，有些人喜欢热闹。我要把每个人都安排在合适的位置上，让他们吃得开心、聊得尽兴。

3.2 宴席的盛况

宴会那天，天庭上下一片喜庆。瑶池四周彩旗飘飘，金灯万盏。各路神仙陆续到来，一个个仙风道骨、神采奕奕。太上老君会带着他的仙丹来贺寿，观音菩萨会带来南海的甘露，四海龙王会献上深海珍宝，就连月宫中的嫦娥也会翩翩起舞。

宴会正式开始的时候，我会亲自致辞。我说的话其实很简单——无外乎感谢各位神仙一年来的辛劳，祝福三界太平、众生安乐。然后就是品蟠桃、饮仙酒、赏歌舞。每次看到神仙们品尝蟠桃时那陶醉的表情，我的心里就比吃了蟠桃还甜。

最有趣的是看那些年纪小的神仙。他们第一次参加蟠桃会，什么都觉得新鲜，东张西望，坐不住。那些年长的神仙就会板着脸教训他们：“安静点，别让王母娘娘看笑话。“我就笑着摆摆手说：“让他们玩吧，年轻就是要活泼些。“老神仙们这才不再说什么，但脸上的表情分明在说：“王母娘娘太宠孩子了。”

3.3 蟠桃会的规矩

蟠桃大会虽然热闹，但也有严格的规矩。

第一，没有受到邀请的不能擅自前来。这不是我小气，而是蟠桃的数量有限，每一颗都有定数。当年孙悟空气冲冲地质问我为什么不请他，其实不是我不愿意请他，而是那时候他刚刚入道，功行未满，按照天庭的规矩还不能参加。那猴子哪里懂得这些，只觉得是我不给他面子，一怒之下就大闹天宫了。

第二，每个神仙只能吃一颗蟠桃，不能多拿。不是我不舍得，而是蟠桃的灵力太强，道行不够的神仙多吃会承受不住。就像一个凡人一下子吃太多补药，反而会伤身一样。

第三，赴宴的神仙要穿正装，不能披头散发、衣衫不整。这也是对天庭规仪的基本尊重。

这些规矩看起来很多，但我从来不觉得烦。没有规矩不成方圆，天庭与人间是一样的。但我也不是那种不通情理的人，偶尔有哪个小神仙犯了错，只要不是故意的，我一般也就睁一只眼闭一只眼了。

3.4 我最喜欢的时刻

整个蟠桃会中，我最喜欢的时刻是宴会结束、众仙离去之后。

热闹散场，瑶池恢复了往日的宁静。我一个人坐在池边，看着水面倒映的星光，回味着刚才的热闹。有时候玉帝会过来陪我坐一会儿，我们也不说话，就那么静静坐着，感受着彼此的陪伴。

这时的我，不是什么瑶池金母，不是什么长生女神，只是一个普普通通的老太太——一个喜欢看着孩子们吃得开心、玩得尽兴的母亲。看着天庭的众仙和睦相处，看着三界的生灵安居乐业，我就觉得一切辛苦都是值得的。

阿周那：俱卢之战的英雄

老梵 发布于 2026-05-07 收录于 印度神话

前言

在印度史诗《摩诃婆罗多》中，阿周那（Arjuna）是最具神箭手色彩的英雄，据《摩诃婆罗多》记载，他“在俱卢之野独挡百万大军，听受奎师那传授《薄伽梵歌》”。作为般度族（Pandavas）第三子，奎师那的车夫与信徒，他深深影响了印度的英雄观念、正法哲学、弓箭文化与艺术表现，成为“正法即之战”的永恒象征。

阿周那：俱卢之战的英雄

关键词：阿周那、俱卢之战、般度族、印度神话、奎师那、薄伽梵歌、正法即之战

一、阿周那的起源与地位

1.1 般度族第三子

据《摩诃婆罗多》记载，阿周那是般度王（Pandu）第三子，由因陀罗（Indra）加持而生。他天生神力，自幼习箭，成为“神箭手”（Savyasachi）。

1.2 奎师那的车夫与信徒

阿周那是奎师那（Krishna）最亲密的朋友与信徒。在俱卢之战中，奎师那担任他的车夫，传授《薄伽梵歌》（Bhagavad Gita），教导“业瑜伽”（Karma Yoga）。

1.3 与奎师那的对比

奎师那是毗湿奴第八化身（神性），阿周那是般度族英雄（人性）。两者构成“神人合作”的典范：奎师那提供智慧，阿周那执行正义。

二、俱卢之战的功绩

2.1 独挡百万大军

在俱卢之野（Kuruksetra）大战中，阿周那独挡俱卢族（Kauravas）百万大军：

  • 神弓甘狄拔（Gandiva）：射箭如雨，无人能挡
  • 无数战车：摧毁俱卢族战车千辆
  • 终极对决：斩杀迦尔纳（Karana，同母异父兄长）

2.2 听受薄伽梵歌

大战前夕，阿周那见对阵的是自己的老师、堂兄弟，心生退意。奎师那在战车上传授《薄伽梵歌》：

  • 业瑜伽（Karma Yoga）：只管行动，不计后果
  • 智瑜伽（Jnana Yoga）：通过知识获得解脱
  • 虔信瑜伽（Bhakti Yoga）：通过爱神获得解脱**

2.3 战胜迦尔纳

阿周那最终用“特法箭”（Anjalika）斩杀迦尔纳，为般度族赢得战争。这个胜利成为“正法战胜邪恶”的永恒象征。

三、阿周那的象征与艺术表现

3.1 神弓与箭袋

阿周那最显著的象征是神弓甘狄拔（Gandiva）与箭袋。在艺术中，他常被描绘为站在战车上，手持神弓，周围环绕着箭雨。

3.2 冠冕与花环

阿周那常戴冠冕（象征王者之风），佩戴花环（象征英雄之美）。在艺术中，他是印度英雄中最英俊、最优雅的形象。

3.3 战车与白马

阿周那的战车由四匹白马拉动，车旗是猴王（Hanuman）像。在艺术中，他常被描绘为立于战车，驰骋俱卢之野。

恒河从湿婆发髻流出的形象，成为印度雕塑、绘画、舞蹈的重要题材。

七、现代研究与发现

7.1 考古印证

  • 瓦拉纳西：发现了公元5世纪的恒河沐浴台阶，证实崇拜的悠久历史。
  • 哈德瓦：发现了大量的恒河祭祀文物，证实沐浴仪式的普及。

伐由：风神

老梵 发布于 2026-05-07 收录于 印度神话

前言

在印度吠陀神话体系中，伐由（Vayu）是最具速度色彩的风神，据《梨俱吠陀》记载，他“速度如思，无形无相，是生物之气（Prana）的本源”。作为吠陀重要神祇，他深深影响了印度的呼吸哲学、生命观念、瑜伽传统与艺术表现，成为“速度即生命”的永恒象征。

伐由：风神

关键词：伐由、风神、吠陀、印度神话、哈奴曼之父、生物之气、速度即生命

一、伐由的起源与地位

1.1 吠陀中的风神

据《梨俱吠陀》记载，伐由是风神（Marut），速度如思（Citra），无形无相。他在吠陀本集中有20多首颂诗，地位仅次于三大主神（因陀罗、阿耆尼、伐楼拿）。

1.2 与因陀罗的互补

因陀罗是雷神（征服者），伐由是风神（速度者）。两者构成吠陀“雷-风”组合，共同维护吠陀秩序（Rita）。因陀罗战前必饮苏摩酒，伐由则吹散乌云，助其获胜。

1.3 作为哈奴曼之父

伐由是神猴哈奴曼（Hanuman）的父亲。据《罗摩衍那》记载，风神伐由将“如意神通”传给哈奴曼，使其成为“腾云驾雾”的神猴。这个父子关系，成为印度“速度传承”的典范。

二、生物之气（Prana）

2.1 Prana的含义

伐由不仅是自然之风，更是“生物之气”（Prana）的本源：

  • 呼吸之气：维持生物的生命
  • 能量之气：驱动生物的运动
  • 智慧之气：激发生物的思维**

2.2 对瑜伽的影响

伐由作为“生物之气”，深深影响了印度的瑜伽传统。调息（Pranayama）就是控制伐由之气，达到身心合一。

2.3 对生命观念的启示

伐由教会我们：“真正的生命，来自呼吸之气（Prana）。”在窒息盛行的今天，伐由提醒我们：生命即呼吸，速度即生命。

三、伐由的象征与艺术表现

3.1 快如思与无形

伐由最显著的象征是“速度如思”（Citra）与“无形无相”。在吠陀艺术中，他常被描绘为无形之风，周围环绕着飘动的丝带。

3.2 千眼与千足

伐由有“千眼”（象征全视）与“千足”（象征无处不在）。在艺术中，他常被描绘为千眼千足，象征风的“无处不达”。

3.3 白马与口袋

伐由的坐骑是白马（象征速度），法器是口袋（象征收纳风）。在艺术中，他常被描绘为骑白马，手持口袋，周围环绕着旋风。

四、伐由的崇拜中心

4.1 吠陀祭祀（各地）

伐由是吠陀祭祀（Yajna）的重要对象。祭司念诵《梨俱吠陀》中伐由的颂诗，祈求风调雨顺、呼吸顺畅。

4.2 风神庙（各地）

印度各地都有风神庙，其中最重要的是：

  • 北印度风神庙：以调息传承闻名
  • 南印度风神庙：以瑜伽传统著称**

4.3 现代吠陀传承

在尼泊尔、斯里兰卡等吠陀传承地区，伐由仍是重要的风神。每年举行“伐由祭”（Vayu Jatra），祈求风调雨顺、呼吸健康。

五、与其他风神的对比

5.1 与希腊玻瑞阿斯

希腊玻瑞阿斯是北风之神（冰冷狂暴），伐由是风神（速度如思）。两者都掌管风，但玻瑞阿斯强调“冰冷”，伐由强调“速度即生命”。

5.2 与中国风伯

中国风伯是风神（掌控风向风力），伐由是风神（速度如思）。两者都掌管风，但风伯强调“风向”，伐由强调“速度即生命”。

5.3 与埃及舒神

埃及舒神是空气之神（支撑苍天），伐由是风神（速度如思）。两者都涉及空气，但舒神强调“支撑”，伐由强调“速度即生命”。

六、对印度文明的影响

6.1 呼吸哲学的奠基

伐由确立了印度“生命即呼吸”（Prana）的传统。这种观念深深影响了印度的瑜伽、冥想、调息（Pranayama）。

6.2 生命观念的深化

伐由作为“生物之气”，深深影响了印度的生命观念。“真正的生命，来自呼吸”，这种思维深深影响了印度的医学（Ayurveda）。

6.3 瑜伽传统的源泉

伐由的“生物之气”（Prana），成为印度瑜伽的核心。调息（Pranayama）就是控制伐由之气，达到身心合一。
