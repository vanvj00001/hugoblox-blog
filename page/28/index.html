杜尔迦最重要的功绩是斩杀魔王摩醯湿：

  • 九天九夜的激战
  • 十臂持兵器，最终用三叉戟刺穿魔王心脏

潘：希腊神话中的牧神与森林之神

老梵 发布于 2026-05-07 收录于 希腊神话

前言

上古之时，希腊山野之间，有神降焉。

神名潘，父为赫尔墨斯，母为德律俄珀，山林女神也。潘初生时，羊角歧趾，遍体绒毛，母见而大骇，呼曰：“怪哉！“遂遁入林，不敢复认。父笑而抱之，献于奥林匹斯之巅。诸神见其状，皆曰：“此小羊孩也，可爱哉！“遂留于神山，为众神之宠儿。

潘稍长，居阿卡迪亚之深林，与宁芙牧人为伴。其状也，上半为人，下半为羊，头有弯角，足有分蹄。每行林间，吹芦苇之箫，七音悠扬，羊群随行，野兽驻足而听。其笛声者，林之声也；其人者，野之魂也。

有牧人问之曰：“汝名为何？“潘答曰：“吾乃潘，万物之灵也。“牧人以其形，半人半羊，乃称之为"潘格洛斯”，意云"全羊”。其名遂传于世。

潘有异能，能于酣睡中骤醒，长啸一声，其声凄厉，穿林越谷，闻者股栗，陷入莫名之惧。公元前四百九十年，波斯王率军来攻雅典，城危在旦夕。忽夜闻潘之啸声，波斯军大恐，以为其神兵天降，遂溃散而逃。雅典人转危为安，自此奉潘为护国之神。

潘创排箫之器，取七芦为之，音韵古朴，为田园之音。相传潘曾慕山林女神绪任克斯之美，追逐不及，女化为芦苇，潘乃取芦制箫，使女之声永留人间。

后基督之光普照大地，有传言曰：“潘死矣。“当其时，异教衰落，神话渐远，而潘之笛声，永留天地之间。

其故事流传不朽。

这是他的故事。

前言

阿波：文莱巨人#

关键词：阿波、文莱神话、巨人专题、婆罗洲守护

一、镇守婆罗洲：山一样大的身体

阿波的身体像基纳巴卢山一样大，每一步踩下去，地面都裂开一道缝。他镇守着婆罗洲，不让外敌入侵。

“这是我的地盘！”阿波吼一声，声音像季风，“谁敢来，我就捏碎他！”

岛上的百姓都敬仰他，每年都来祭拜，求他保佑航运平安。

二、与殖民者斗法：魔法的较量

英国殖民者带着火炮来了，阿波喷出毒火，想把他们的船烧了。

“这怪物，太可怕了！”殖民者吓得后退，“用《古兰经》，只有真主的经文能破他的魔法！”

伊玛目念着《古兰经》，把经文洒向阿波。没想到，经文一碰到他的皮肤，他就安静了下来。

三、皈依伊斯兰教：成为守护神

伊玛目对阿波说：“你若皈依伊斯兰教，我就封你为婆罗洲的守护神。”

阿波叩首：“弟子愿皈依。”

从此，阿波成了婆罗洲的守护神。渔民们出海前，都会来祭拜他，说：“阿波保佑我们平安。”

四、巨人的 Legacy：基纳巴卢的传说

阿波死后，他的身体化作了基纳巴卢山。直到今天，文莱人还说：“那座山，就是阿波的化身。”

他的故事，成了文莱神话里“从反抗到皈依”的经典，也记录了伊斯兰教传入文莱的历史。

五、总结

阿波作为文莱神话里的巨人，用山一样的身体镇守婆罗洲，最终皈依伊斯兰教成为守护神。他的故事，既是文莱民族的守护传奇，也是“从反抗到融合”的宗教寓言，提醒我们：真正的强大，来自包容与信仰。

结为夫妻

  • 婚后开药铺保和堂，白素贞以医术济世救人
  • 夫妻恩爱，生活美满，不久白素贞怀孕
  • 许仙始终不知妻子是千年白蛇精

端午现形

  • 端午节，许仙误让白素贞饮下雄黄酒
  • 白素贞现出白蛇原形，许仙惊吓而死
  • 白素贞冒死上昆仑山盗取仙草，救活许仙
  • 许仙复活后，虽知真相，仍深爱白素贞

三、与法海的对抗

法海是白素贞命运中最强大的对手，代表着佛法与秩序的压制。

法海出场

  • 金山寺住持法海，奉佛法旨，认为人妖殊途
  • 识破白素贞白蛇身份，决心收服她
  • 设计让许仙上金山寺进香，强行扣留
  • 告诉许仙白素贞是蛇精，使其心生恐惧

水漫金山

  • 白素贞为救许仙，与小青水漫金山寺
  • 召集虾兵蟹将、长江水族，水势滔天
  • 法海以袈裟、金钵等法器对抗，护住金山寺
  • 白素贞因怀孕法力减弱，最终败退

镇压雷峰

  • 法海用金钵收服白素贞，镇压于雷峰塔下
  • 塔下留字：“西湖水干，江湖不起，雷峰塔倒，白蛇出世”

  • 发誓要练成绝世武功，推倒雷峰塔救出姐姐

现代演绎

本主：白族村寨保护神

老梵 发布于 2026-05-07 收录于 中国西南神话

前言

本主是白族特有的保护神信仰，每个白族村寨都有自己专属的本主，被尊为村寨的守护神、纠纷仲裁者、丰收赐予者。本主既可以是神话人物（如猎神杜朝选），也可以是历史人物（如大理国开国皇帝段思平），还可以是自然物（如大树、石头），体现了白族信仰的多元融合特征。本主信仰是白族文化认同的核心，渗透到村寨生活的方方面面。本文将系统梳理本主的起源、神格特征、多元来源、祭祀仪式及文化影响，揭示这一独特信仰如何成为白族社会的精神支柱。

本主：白族村寨保护神

关键词：本主、白族神话、村寨保护神、段思平、杜朝选、多元融合、本主节、西南神话

一、本主的起源与神格

本主信仰起源于白族先民的祖先崇拜与自然崇拜，后逐渐发展为独特的村寨保护神体系。

信仰起源

  • 最早源于白族先民的祖先崇拜，将氏族首领尊为保护神
  • 融入自然崇拜：山神、水神、树神皆可成为本主
  • 受汉族影响：历史人物（段思平、李宓）被纳入本主体系
  • 受藏族、傣族影响：部分本主带有藏传佛教、南传佛教色彩

神格定位

  • 村寨保护神：守护村寨平安、人畜兴旺、五谷丰登
  • 纠纷仲裁者：村寨内部纠纷由本主裁决
  • 丰收赐予者：保佑农业生产，赐予丰收
  • 多元神格：兼具神话人物、历史人物、自然物三重属性

二、本主的多元来源

本主最显著的特征是来源多元，同一村寨的本主可能完全不同。

神话人物本主

  • 杜朝选：猎神，传说他射杀九头魔王，拯救白族百姓
  • 柏洁夫人：坚贞不屈的女性，被尊为大理地区本主
  • 张小三：传说中的英雄，为白族人民治水有功
  • 这些本主多为白族原生神话人物，年代久远

历史人物本主

  • 段思平：大理国开国皇帝，被许多村寨尊为本主
  • 李宓：唐朝将领，征讨南诏时战死，被尊为本主
  • 郑回：南诏国清平官，辅佐南诏王有功
  • 这些本主反映白族与中原王朝的历史互动

自然物本主

  • 神树：大理白族将大青树尊为本主
  • 石头：剑川白族将奇石尊为本主
  • 龙王：洱海周边村寨将龙王尊为本主
  • 这些本主保留最原始的自然崇拜痕迹

三、著名本主举例

白族不同地区有自己独特的本主，其中最著名的有以下几位。

段思平（大理国开国皇帝）

  • 被大理地区众多村寨尊为本主
  • 传说他建立大理国后，轻徭薄赋，深得民心
  • 本主庙中，段思平塑像身着帝王服饰，威严庄重
  • 每年本主节，段思平本主庙香火最盛

杜朝选（猎神）

  • 被剑川、鹤庆等地村寨尊为本主
  • 传说他射杀九头魔王，拯救白族百姓于水火
  • 猎神形象手持弓箭，身披兽皮，勇猛威武
  • 猎户特别崇拜杜朝选，出猎前必祭

柏洁夫人

  • 被大理、洱源等地村寨尊为本主
  • 传说她丈夫被南诏王杀害，她坚贞不屈，跳海自尽
  • 被尊为贞洁女神，妇女特别崇拜
  • 本主庙中，柏洁夫人塑像端庄慈祥

四、本主祭祀与节庆

对本主的祭祀是白族村寨最重要的宗教活动。

本主节

  • 每个村寨有自己的本主节，日期各不相同
  • 节日期间，全村出动，抬本主塑像巡游村寨
  • 举行耍龙、舞狮、唱戏、赛龙舟等活动
  • 杀猪宰羊，宴请亲友，热闹非凡

日常祭祀

  • 村寨入口处建有本主庙，塑像端坐其中
  • 村民每日清晨到本主庙敬香、献供品
  • 逢年过节、婚丧嫁娶、盖新房，都须祭拜本主
  • 遇到灾难疾病，请本主庙主持念经祈福

祭祀仪式

  • 本主节由村寨长老主持，仪式隆重
  • 杀猪宰羊，献上五谷、水果、美酒
  • 念诵本主经，追溯本主功绩，祈求护佑
  • 全村聚餐，共享祭品，增进村寨凝聚力
