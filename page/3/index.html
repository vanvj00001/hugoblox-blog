前言

北宋年间的蓬莱阁庙会，挤满了赶海的百姓。一个穿破道袍的老道士坐在石阶上，围坐着七八个光着脚的小孩，有的揪他的白胡子，有的摸他腰间挂的破葫芦，叽叽喳喳问个不停。他手里转着个豁口葫芦，笑得皱纹都挤到了一块儿：“今天给你们讲讲八仙的故事——这八位神仙啊，有瘸着腿拄铁拐的乞丐，有倒骑毛驴的老头，有穿破蓝衫一脚鞋一脚赤的浪子，有拿荷花的农家女，还有唯一的女仙，他们凑在一起，闹出了八仙过海的大动静，连龙王都得给他们赔礼。”

本文将用故事性的笔触，带你走进八仙的世界：从他们各自离奇的成仙经历，到八位神仙凑成一伙的缘由，再看他们一起过海时引发的龙宫风波，理解中国民间对“和而不同”的朴素向往。

一、八仙的由来：从散仙到群体

八仙的说法最早出现在唐宋的笔记里，起初只是零散的神仙传说，没有固定的名单。唐朝《太平广记》里记载的“八仙”还有刘海，并没有何仙姑和曹国舅；到了北宋，坊间话本里才慢慢把蓝采和、张果老固定进来；直到明朝吴元泰的《东游记》，才正式定型为铁拐李、汉钟离、张果老、蓝采和、何仙姑、吕洞宾、韩湘子、曹国舅八位，把刘海换成了曹国舅，才有了我们现在熟知的八仙名单。

他们出身各异：铁拐李是终南山落第书生，汉钟离是汉朝败军之将，张果老是江湖游方的术士，蓝采和是街头卖唱的浪子，何仙姑是广州增城采荷的农家女，吕洞宾是屡试不第的书生，韩湘子是韩愈府上的纨绔侄子，曹国舅是宋仁宗的皇亲国舅。老道士摸着小孩的头说：“你看，不管你是什么出身，是乞丐还是国舅，只要心向善，都能成仙——这就是老百姓最爱八仙的原因，他们不是高高在上的天庭正神，是和我们一样的普通人变的。”

有次个小娃子问：“那刘海为啥被换下去了呀？”老道士哈哈大笑：“刘海那是后来的财神，和这八位凑不到一块儿去，曹国舅本来就是皇亲，换进去刚好凑成八个，刚好能坐满一张八仙桌，你说巧不巧？”

二、各自的绝活：各显神通的底气

八仙每人都有一件专属法宝，还有独到的本事，过海的时候全靠这些吃饭： 铁拐李的铁拐是太上老君亲赐的，能当船浮在水上走，背上的葫芦里装的是能治百病的仙丹，上次蓬莱村有个孕妇难产，他倒出一颗丹药，孕妇立马就顺产了； 汉钟离的芭蕉扇是东华帝君给的，扇一下能灭三千丈的火，扇两下能唤醒断了气的人，扇三下能让枯树发新芽，上次终南山山火，他扇了两下就把火全灭了； 张果老的毛驴是仙兽，不用吃草，喷一口仙气就能日行千里，还能折叠成巴掌大的驴皮，塞进口袋里，上次他驮了十个小孩去赶庙会，驴腿都没打颤； 蓝采和的花篮是竹编的，看着破，其实能装下整片大海，唱的歌能引得蝴蝶跟着飞，上次他在海边唱了半天，引来了上百只蝴蝶围着花篮转； 何仙姑的荷花是清晨在荷塘采的，摘一片就能预知未来吉凶，发红就是好事，发黑就是灾祸，上次村里后生要出门做生意，她给了片花瓣，发黑，后生绕了路，刚好躲过了山匪； 吕洞宾的纯阳剑是汉钟离所赠，能斩妖除魔，三醉岳阳楼度化了无数凡人，上次他在汴梁街头，一剑斩了个祸害百姓的妖道，百姓都跪着喊活神仙； 韩湘子的箫是紫竹做的，吹一曲能引来凤凰，吹两曲能让枯木一夜开花，吹三曲能让死人坐起来，上次他在韩愈府上吹了一曲，院子里的枯梅一夜开了满树的花； 曹国舅的玉板是羊脂玉做的，敲一下就能净化人的心灵，消除人的罪孽，上次他碰到个抢了百姓钱的贪官，敲了一下玉板，贪官立马跪在地上，把抢的钱全还了回去。

老道士说着，从葫芦里倒出一颗糖塞给小孩：“当初他们过海，就是靠这些本事，没用一条船，你说厉害不厉害？”

三、凑成一伙：从单干到结伴

八仙原本都是散仙，各修各的道，谁也不挨谁。直到有一天，吕洞宾在岳阳楼喝酒，碰到汉钟离也在那儿喝，两人聊了半天，发现脾性特别合，吕洞宾说：“我这人最爱喝酒，遇到你这老道，正好搭伴云游。”汉钟离笑着说：“我正愁没人一起云游，你这书生倒是合我脾气。”

两人结伴走到终南山，碰到铁拐李正蹲在路边给小孩治冻疮，铁拐李瘸着腿，葫芦里的仙丹香得小孩都围着他转。吕洞宾喊他：“老李，跟我们一起走呗，人多热闹。”铁拐李把葫芦往腰上一挂：“走啊，我正愁没人一起喝酒呢！”三个人又走到长安街头，碰到张果老倒骑着驴，正跟小孩说“回头看才懂前路”，蓝采和挎着花篮，一脚鞋一脚赤，正扯着嗓子唱“踏歌蓝采和，世界能几何”，听说要组队，蓝采和把花篮往背上一甩：“走啊，我正愁没人一起玩呢！”

后来他们走到广州增城的荷塘边，碰到何仙姑正采荷花，何仙姑笑着说：“我跟你们走，正好给你们采荷花当点心。”走到韩愈府上，碰到韩湘子正拿着箫吹曲，韩湘子说：“我叔父总骂我不务正业，我跟你们走了，他倒省心。”最后在汴梁的城门口，碰到曹国舅穿着官服辞官出来，曹国舅叹气：“这官我当够了，跟你们一起修道去。”八个人终于凑齐了。老道士拍着大腿：“你说巧不巧？八个人，刚好凑成一桌，刚好能闹出八仙过海的大动静，这是天定的缘分啊。”

四、一起的故事：八仙过海的风波

八仙凑齐后，在蓬莱阁聚会喝酒，桌上摆着从各地搜罗的好酒，铁拐李的葫芦酒，汉钟离的芭蕉扇扇着风，张果老的驴驮着酒坛，蓝采和唱着歌，何仙姑摆着荷花，吕洞宾提着剑，韩湘子吹着箫，曹国舅敲着玉板，热闹得不行。

吕洞宾喝得满脸通红，拍着桌子说：“咱们都是神仙了，下次过海，别用船，各显神通，怎么样？”七个人齐声叫好。张果老倒骑着驴说：“我倒骑驴走，回头看海景。”蓝采和挎着花篮说：“我的花篮能飘在海面上，还能装海鲜。”何仙姑拿着荷花说：“我的荷花能浮水，还能遮太阳。”铁拐李拄着铁拐说：“我的铁拐当船，稳得很。”汉钟离摇着芭蕉扇说：“我的扇子能扇风，推着我走。”吕洞宾提着纯阳剑说：“我的剑能斩浪，开路！”韩湘子拿着箫说：“我的箫声能引路，跟着我走。”曹国舅敲着玉板说：“我的玉板能平浪，保平安。”

结果蓝采和的花篮刚飘到海中央，就蹿出个穿金甲的龙王太子敖丙，一把抢了过去：“这花篮不错，归我了！”蓝采和急得在花篮上跳脚：“还我花篮！还我花篮！”八仙大怒，跟着闯进龙宫，铁拐李用铁拐打翻了虾兵蟹将，汉钟离用芭蕉扇扇得龙宫摇摇晃晃，吕洞宾用纯阳剑斩断了龙宫的水晶柱子，何仙姑用荷花挡住了龙宫的毒水，韩湘子用箫声引来了海里的鲸鱼，曹国舅用玉板净化了龙宫的邪祟，打了整整三天三夜，把龙王的水晶宫都震裂了。最后龙王亲自出面，赔着笑脸说：“各位神仙，是我儿子不懂事，我赔礼了，花篮还你们，还派虾兵蟹将给你们清路。”八仙这才作罢。

老道士说到这儿，指着海面：“你看，那边的浪，就是当年八仙闹出来的。现在老百姓还说‘八仙过海，各显神通’，就是讲他们靠自己的本事，闯出一片天，不用靠天靠地，靠自己就行。”

大洪水神话的分布范围极广，几乎覆盖了所有人类文明发源地。

共性特征

  • 记录：《淮南子》《列子》等典籍详细记载
  • 特点：与女娲补天、大禹治水共同构成华夏治水文明叙事

  • 起因：尼罗河每年定期泛滥，并非神怒引发的灭世洪水

  • 特点：唯一将洪水视为恩赐而非惩罚的主要文明

美索不达米亚：神怒型灭世洪水

  • 特点：方舟叙事最早的文字记录，直接影响基督教诺亚方舟故事

基督教：惩戒型灭世洪水

  • 扎拉莫（Zaramo）：雨水与丰收之神，掌管农业。

二、基库尤神话：恩柴与创世

肯尼亚基库尤族的神话以恩柴为核心：

  • 恩柴（Ngai）：最高神，居住在肯尼亚山，掌管雨水、丰收与生命。
  • 莫鲁（Muru）：恩柴的使者，掌管雷电与风暴。
  • 瓦里穆（Wariimu）：祖先精灵，保佑后代。

三、马赛神话：恩加伊与牛群

坦桑尼亚、肯尼亚马赛族的神话与牛群紧密相关：

  • 恩加伊（Engai）：最高神，分为黑色恩加伊（仁慈）与红色恩加伊（愤怒）。
  • 纳帕希（Napaei）：马赛部落的保护神，掌管战争与狩猎。

四、阿赞德神话：巫术与魔法

刚果盆地东部的阿赞德族神话以巫术为核心：

  • 姆班加（Mbanza）：最高神，创造人类与自然。
  • 巫师（Nganga）：掌握魔法，能治病或施咒。
