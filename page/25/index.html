埃及神话人物关系考：从创世九神到冥界审判的谱系网络

老梵 发布于 2026-05-07 收录于 埃及神话

前言

埃及神话是世界上最古老的神话体系之一，起源于公元前3000年左右的尼罗河流域。其神系以九神团（Ennead）为核心，从原初之水努恩（Nun）中诞生的创世神，到奥西里斯-伊西斯-荷鲁斯的神圣家庭，再到阿努比斯的木乃伊制作与冥界审判，埃及神话展现了一个秩序（Ma’at）与混沌（Isfet）永恒斗争的宇宙观。本文将系统梳理从创世到冥界审判的核心人物关系，揭示埃及神话"创世-死亡-重生"的循环架构。

埃及神话人物关系考：从创世九神到冥界审判的谱系网络

关键词：埃及神话、人物关系、拉、伊西斯、奥西里斯、荷鲁斯、赛特、阿努比斯、托特、九神团、冥界审判、玛特、来世

一、创世神与九神团（Ennead）

埃及神话的创世始于原初之水努恩：

  • 拉（Ra）：太阳神，从努恩中诞生，乘坐太阳船穿越天空与冥界
  • 舒（Shu）：空气之神，拉之子，支撑天空
  • 泰芙努特（Tefnut）：湿气女神，拉之女，舒的姐妹兼妻子
  • 盖布（Geb）：大地之神，舒与泰芙努特之子
  • 努特（Nut）：天空女神，舒与泰芙努特之女，盖布的姐妹兼妻子
  • 奥西里斯（Osiris）：冥王，盖布与努特之子
  • 伊西斯（Isis）：生命与魔法女神，盖布与努特之女，奥西里斯的姐妹兼妻子
  • 赛特（Set）：混乱与风暴之神，盖布与努特之子，奥西里斯的兄弟
  • 奈芙蒂斯（Nephthys）：死者的守护者，盖布与努特之女，赛特的姐妹兼妻子

相传在永恒之初，原初之水努恩（Nun）无边无际，混沌一片。突然，一座发光的神山从水中升起，太阳神拉从莲花中破茧而出，他的第一声啼哭便是世界的开端。拉睁开双眼，光芒照亮了原本漆黑的宇宙，他注视着这片虚空，说道：“我要创造万物。”于是，他用意念创造了空气之神舒与湿气女神泰芙努特，又让舒与泰芙努特结合，生下了大地之神盖布与天空女神努特。当盖布与努特紧紧相拥时，拉命令舒将努特高高托起，从此天空与大地分离，世界初具雏形。九神团的其他成员也相继诞生，奥西里斯、伊西斯、赛特、奈芙蒂斯，构成了埃及神话的核心神系。

拉神在创造世界后，感到孤独，便从自己的眼泪中创造了人类。他流下一滴眼泪，落在尼罗河的泥土中，变成了第一个人类。拉神赋予人类生命与智慧，让他们统治大地。然而，人类后来变得傲慢，不再敬畏神明，拉神便派女神哈托尔（Hathor）去惩罚人类。哈托尔化身为狮头女战神塞赫麦特（Sekhmet），疯狂屠杀人类，直到拉神后悔，用啤酒染红让哈托尔误以为是血，从而喝醉停止杀戮。这个故事解释了人类的不完美与神的慈悲。

二、奥西里斯神圣家庭

奥西里斯-伊西斯-荷鲁斯三角

  • 奥西里斯：被兄弟赛特杀害并分尸，后被伊西斯复活，成为冥王
  • 伊西斯：最伟大的魔法女神，收集奥西里斯的尸体并使其复活，生下荷鲁斯
  • 荷鲁斯（Horus）：鹰神，奥西里斯与伊西斯之子，与赛特争夺王位获胜

奥西里斯是一位仁慈的统治者，他教会人类耕种、酿酒、制定法律，被百姓尊为“文明传播者”。然而，他的兄弟赛特嫉妒他的威望，设计了一场致命的阴谋。在一次宴会上，赛特命人制作了一口精美的水晶棺材，声称要送给身材最匹配的人。当奥西里斯躺入棺材时，赛特立即盖上盖子，用铅水密封，将棺材抛入尼罗河。伊西斯得知噩耗后，悲痛欲绝，她剪下自己的一缕头发，穿上丧服，发誓要找回丈夫的尸体。她沿着尼罗河跋涉千里，终于在腓尼基的海岸找到了被冲上岸的棺材。然而，赛特发现了尸体，将其切成十四块，散落各地。伊西斯不辞辛劳，一块一块地收集，用魔法将奥西里斯短暂复活，生下了鹰神荷鲁斯。荷鲁斯长大后，向赛特发起挑战，这场持续八十年的王位之争，最终以荷鲁斯的胜利告终。

伊西斯在收集奥西里斯的尸体时，遇到了许多困难。有些尸体部位被鱼吃掉，有些被埋在地下深处。她每找到一块，就用魔法包裹起来，制成木乃伊。最终，她找到了除生殖器外的所有部位（生殖器被螃蟹吃掉）。她用魔法创造了替代品，将奥西里斯的尸体完整复原，从而让他短暂复活，生下了荷鲁斯。

在一次战斗中，荷鲁斯挖出了赛特的睾丸，而赛特则挖出了荷鲁斯的眼睛。这就是著名的“荷鲁斯之眼”（Eye of Horus）的由来。托特神用魔法治好了荷鲁斯的眼睛，这只眼睛后来成为埃及的保护符号，代表着牺牲与重生。

三、冥界与审判体系

阿努比斯（Anubis） 胡狼头神，木乃伊制作与防腐之神，引导死者前往冥界，主持"称心仪式"（心脏与玛特羽毛称重）。

托特（Thoth） 朱鹭头神，智慧、文字、月亮之神，记录冥界审判结果，协助伊西斯施展魔法。

玛特（Ma’at） 真理与正义女神，羽毛作为冥界审判的标准，代表宇宙秩序。

阿努比斯是冥界的引路人，他的胡狼头像象征着死亡与防腐。当一个人死后，阿努比斯会引导其灵魂前往冥界，进行木乃伊制作，确保尸体能在来世完好保存。最关键的仪式是“称心”：阿努比斯将死者的心脏放在天平的一端，另一端放上玛特女神的真理羽毛。如果心脏比羽毛轻，说明死者一生行善，可以进入芦苇地享受永生；如果心脏比羽毛重，说明死者罪孽深重，将被恶魔阿米特吞噬。托特神则在一旁记录审判结果，用他的朱鹭笔在生死簿上写下判决书。玛特女神不仅是审判的标准，更是宇宙秩序的守护者，她的羽毛代表着真理、正义与平衡，是埃及社会道德的核心准则。

四、拉与太阳神信仰

拉的宇宙航行 拉每天乘坐太阳船（Mandjet）穿越天空，夜晚进入冥界（杜亚特），与巨蛇阿佩普（Apep）战斗，次日重生。

拉与法老的关系 法老被视为荷鲁斯在人间化身，死后成为奥西里斯。

拉每天清晨从东方地平线升起，乘坐太阳船“曼杰特”（Mandjet）穿越天空，将光明与温暖带给大地。然而，每当夜幕降临，拉进入冥界（杜亚特），他必须面对巨蛇阿佩普（Apep）的袭击。阿佩普是混沌的化身，身体长达数百尺，浑身漆黑，盘踞在冥界的暗河中。每晚，拉的太阳船都会遭遇阿佩普的猛烈攻击，船员们用咒语和武器与巨蛇搏斗。最终，拉总能战胜阿佩普，在次日清晨重生，再次从东方升起。这种昼夜循环象征着秩序与混沌的永恒斗争，也预示着法老的神圣地位。法老被视为荷鲁斯在人间化身，他统治的每一个决定都代表着玛特女神的意志。当法老死去，他的灵魂将升入天界，成为奥西里斯，在冥界继续守护着埃及的秩序。

五、神族关系网络总结

埃及神话人物关系可归纳为三大网络：

  1. 创世网络：努恩 → 拉 → 九神团 → 宇宙秩序
  2. 神圣家庭网络：奥西里斯 + 伊西斯 → 荷鲁斯 → 王位传承
  3. 冥界审判网络：阿努比斯 + 托特 + 玛特 → 来世审判

这三大网络共同构成了埃及神话"创世-死亡-重生"的循环体系，深刻影响了古埃及的宗教、艺术与金字塔文明。

埃及神话不仅塑造了古埃及人的日常生活，更深刻影响了他们的建筑、艺术和丧葬习俗。金字塔内的经文（Pyramid Texts）记载了大量神话片段，这些经文被刻在金字塔的墙壁上，指引法老在来世的旅程。木乃伊制作技术源于奥西里斯重生的传说，伊西斯用魔法复活奥西里斯的故事，让古埃及人相信尸体保存是来世重生的关键。埃及神话还影响了后来的希腊罗马神话，希腊人将拉与赫利俄斯（Helios）对应，将奥西里斯与狄俄尼索斯（Dionysus）联系，这种文化融合体现了埃及神话的深远影响力。

伊西斯寻找奥西里斯尸体的过程充满了艰辛与奇迹。她化身为普通农妇，沿着尼罗河岸跋涉，每到一处就询问当地居民是否见过一口水晶棺材。她风餐露宿，历经数月，终于在腓尼基的比布鲁斯（Byblos）城，看到一棵巨大的柽柳树中长出了一口棺材。原来，奥西里斯的棺材被海浪冲上岸，被柽柳树的根包裹起来，被当地国王砍下做了宫殿的柱子。伊西斯以保姆的身份进入王宫，用魔法让小王子生病，借此要求国王允许她治疗，从而发现了柱子中的棺材。她用魔法将棺材带回埃及，隐藏在尼罗河三角洲的沼泽中。

荷鲁斯与赛特的王位之争持续了八十年，期间发生了许多精彩的较量。赛特曾提议谁能在尼罗河上潜水更久谁就获胜，结果荷鲁斯遵守规则，而赛特却变成河马潜入水底长达数月。伊西斯看不下去，用绳子套住赛特，赛特愤怒地割掉了伊西斯的头颅。后来，众神召开法庭，托特神用魔法帮伊西斯长出了新的头颅。最终，荷鲁斯用渔网捕获了赛特，并在众神面前讲述了赛特的种种恶行。拉神判定荷鲁斯获胜，赛特被流放到沙漠，成为风暴之神。

这些神话故事是古代文明的瑰宝，它们穿越了千年的时光，至今仍在艺术、文学、影视中焕发着生机。从创世神的壮举到英雄的冒险，从神魔的争斗到凡人的智慧，每一个情节都承载着古人对世界的理解与想象。它们教会了我们什么是勇气、什么是智慧、什么是爱、什么是牺牲，成为人类共同的精神财富。 （全文约3123字）

安丹：非洲班图巨人

前言

在非洲班图神话里，安丹是草原上的巨人，脚掌像芭蕉叶一样大，能举起整座山脉当武器。本文将用故事性的笔触，带你走进这位非洲巨人的传奇：从他在大草原上的行走，到众神用蜂蜜酒灌醉他，再看这位巨人的末日，理解非洲人对自然力量的敬畏。

安丹：非洲班图巨人#

关键词：安丹、非洲神话、班图神话、巨人专题、草原巨人

一、草原上的巨足：每一步都震裂大地

安丹的脚掌像芭蕉叶一样大，每一步踩下去，草原上就多了一个深坑。他弯腰捡起一座山脉，当棍子挥舞，把天上的云都扫散了。

“这草原太小了，”安丹嘟囔着，“得找点更大的地方。”

他的汗水流下来，变成了赞比西河；他的呼吸吹过，草原上的草木都弯了腰；他的影子投在地上，遮住了整个村庄。

二、众神的恐惧：草原的威胁

众神在天上看着安丹，心里都发毛：“这巨人要是把整个非洲都踩碎了，我们住哪里？”

天神穆伦圭对众神说：“得想办法杀了他，不然草原永无宁日。”

可安丹太强了，徒手能撕开狮子，一口能吞下一头大象。众神商量了三天三夜，终于想出一条计策：“用蜂蜜酒灌醉他！”

三、蜂蜜酒的陷阱：巨人的贪杯

众神派了最美的女神涅尼，带着十桶蜂蜜酒去找安丹。

“伟大的巨人，”涅尼笑着说，“这是天神特酿的蜂蜜酒，请你品尝。”

安丹眼睛都亮了：“好香的酒！”他一口干了第一桶，又干了第二桶，一口气喝了八桶，终于醉得东倒西歪。

“再来……再来……”安丹嘟囔着，轰然倒地，鼾声震得草原上的鸟儿都飞了。

四、巨人的末日：众神的围杀

众神看到安丹醉倒，纷纷拿出武器冲了下来。

穆伦圭用雷锤砸他的脑袋，铁匠神用烧红的铁钎刺他的眼睛，草原神用藤蔓绑住他的手脚。

安丹在梦里挣扎：“谁……谁在打我……”可蜂蜜酒太烈了，他连眼睛都睁不开，就被众神活活打死了。

他的尸体倒在地上，变成了非洲的第二个高原；他的血液变成了红土，滋养了这片大地。

五、总结

安丹作为非洲班图神话里的草原巨人，用巨大的脚掌震裂大地，最终被众神用蜂蜜酒灌醉后杀死。他的故事，既是非洲人对自然力量的敬畏，也是“再强大的巨人，也敌不过智慧与阴谋”的寓言，提醒我们：力量再大，也抵不过策略与协作。

安格拉·曼纽：波斯神话中的恶神

前言

安格拉·曼纽是波斯神话（琐罗亚斯德教）中的恶神，与善神阿胡拉·马兹达同时诞生，代表黑暗、谎言与灾难。他是善神的最终对手，是波斯宗教中善恶二元对立的永恒象征。

安格拉·曼纽：波斯神话中的恶神#

关键词：安格拉·曼纽、恶神、波斯神话、琐罗亚斯德教、与善神对立、最终被消灭

一、安格拉·曼纽的身世与地位#

1.1 与阿胡拉·马兹达同时诞生#

安格拉·曼纽（Angra Mainyu）与善神阿胡拉·马兹达（Ahura Mazda）同时诞生，但选择黑暗与谎言的道路。他是宇宙恶的源头，与善神永恒对立。

1.2 恶神与黑暗的象征#

安格拉·曼纽是黑暗、谎言、灾难与死亡的象征。他创造毒物、恶兽、疾病，引诱人类远离正道，是波斯宗教中一切恶的源头。

1.3 与善神的对立#

安格拉·曼纽与阿胡拉·马兹达是永恒的对立者。善神代表光明、真理、秩序；恶神代表黑暗、谎言、混乱。两者的斗争，构成琐罗亚斯德教的核心宇宙观。

二、安格拉·曼纽的象征与力量#

2.1 黑暗与灾难#

安格拉·曼纽的象征物是黑暗（恶的温床）、毒物（灾难之源）。他的存在，让波斯人相信：黑暗与恶是永恒的，但通过善行终将战胜。

2.2 引诱人类作恶#

安格拉·曼纽擅长引诱人类作恶，让人类远离正道（Asha）。他通过谎言、贪婪、暴力，让人类陷入痛苦与混乱。

2.3 最终将被消灭#

预言中，安格拉·曼纽最终将被善神阿胡拉·马兹达消灭。他的黑暗终将消散，光明终将普照大地，这是琐罗亚斯德教的核心信仰。

三、安格拉·曼纽的神话#

3.1 创造恶兽与毒物#

安格拉·曼纽作为波斯神话中的恶神，以其黑暗、谎言与最终被消灭的预言，成为波斯宗教中善恶对立的永恒象征。他的故事告诉我们：恶是永恒的，但通过善行终将战胜。

八岐大蛇：日本八头巨蛇

前言

在日本神话里，八岐大蛇是八头八尾的巨蛇，每年要吃一个女孩，须佐之男用酒灌醉它，砍断尾巴后得到天丛云剑。本文将用故事性的笔触，带你走进这场斩蛇传奇：从巨蛇的诅咒，到须佐之男设下酒阵，再看八桶烈酒如何灌醉巨蛇，最后天丛云剑的出世。

八岐大蛇：日本八头巨蛇#

关键词：八岐大蛇、须佐之男、日本神话、巨人专题、天丛云剑

一、巨蛇的诅咒：肥河的血雾

八岐大蛇的身体像山脉一样盘在肥河床上，八个头轮流喷出毒雾，把河边的树木都枯死了。它的眼睛像八个燃烧的火球，鳞片比盾牌还硬。

“今年又到了吃人的日子！”八岐大蛇的八个头同时咆哮，“老爷爷，把你女儿交出来！”

老夫妇哭着躲在家里：“这怪物已经吃了我们七个女儿，只剩下一个了！”

须佐之男正好路过，看到这惨状，心里升起一股怒火：“这巨蛇，我今天就要斩了你！”

二、须佐之男的计策：八桶烈酒

须佐之男让老夫妇造了八个高台，每个台上放一桶烈酒：“等巨蛇来喝酒，它的八个头会各占一桶，喝醉后就动弹不得。”

他又在河边设下篱笆，绑上八根粗绳，每根绳上挂满诱饵：“巨蛇一醉，我就用这些绳子绑住它的八个头！”

八岐大蛇如期而至，八个头同时冲向酒桶：“好香的酒！今天先喝酒，再吃女孩！”

前言

八仙凑齐后，最轰动的事就是“八仙过海”。本文将用故事性的笔触，带你走进这场传奇：从蓬莱阁的喝酒提议，到蓝采和花篮被抢，再看八仙大闹龙宫，最后龙王出面和解，留下“八仙过海，各显神通”的千古谚语。

八仙过海：各显神通

关键词：八仙过海、八仙、八仙专题

一、蓬莱的提议：不用船过海

八仙在蓬莱阁聚会喝酒，正值庙会，阁外挤满了赶海的百姓，海风裹着咸味和饭香吹进来。桌上摆着从各地搜罗的好酒：铁拐李的葫芦酒烈得呛人，汉钟离的芭蕉扇扇得酒香飘满阁，张果老的驴驮着三坛陈酿，蓝采和的花篮里还装着刚摘的野果，何仙姑的荷花摆在桌中央，花瓣上的露珠滚来滚去，吕洞宾的纯阳剑靠在桌边泛着金光，韩湘子的紫箫横在膝头，曹国舅的玉板敲得“叮叮”响。

吕洞宾喝得满脸通红，酒洒在青布道袍上，拍着桌子说：“咱们都是神仙了，下次过海，别用船，各显神通，怎么样？”七个人齐声叫好，声音大得把阁外的海鸥都惊飞了。

铁拐李瘸着腿站起来，铁拐敲在木桌上“当当”响：“我的铁拐能当船，浮在水上走，稳得很！”汉钟离摇着芭蕉扇：“我的扇子能扇风，推着我走，还能扇灭海上的浪。”张果老倒骑着驴，驴鼻子喷着紫色的仙气：“我的驴不用吃草，喷口仙气就走，还能驮三坛酒。”蓝采和挎着花篮，一脚鞋一脚赤：“我的花篮能飘在海面上，还能装海鲜，上次装了三斗螃蟹呢！”何仙姑拿着荷花：“我的荷花能浮水，还能遮太阳，不怕晒。”吕洞宾提着纯阳剑：“我的剑能斩浪，开路！”韩湘子拿着紫箫：“我的箫声能引路，跟着我走。”曹国舅敲着玉板：“我的玉板能平浪，保平安。”

二、花篮被抢：龙宫风波

八仙说干就干，各自施展本事往海对面走。蓝采和的花篮刚飘到海中央，海浪打在花篮上“啪啪”响，他正哼着小曲，突然蹿出个穿金甲的龙王太子敖丙，金甲闪着刺眼的光，一把抢了花篮：“这花篮不错，归我了！”

蓝采和急得在花篮上跳脚，一脚鞋都掉了：“还我花篮！还我花篮！那是我吃饭的家伙！”敖丙哈哈大笑，转身就往海里钻，蓝采和差点跟着掉下去，幸亏路过的渔夫拉了他一把。

八仙大怒，跟着闯进龙宫。铁拐李用铁拐打翻了虾兵蟹将，铁拐敲在珊瑚兵器上“铛铛”响；汉钟离用芭蕉扇扇得龙宫摇摇晃晃，水晶柱子都晃出了裂缝；吕洞宾用纯阳剑斩断了龙宫的三根水晶柱，剑气把龙王的玉桌都劈成了两半；何仙姑用荷花挡住了龙宫喷出的毒水，荷花的花瓣连湿都不湿；韩湘子用箫声引来了海里的鲸鱼，鲸鱼甩着尾巴把龙宫的围墙都撞塌了；曹国舅用玉板净化了龙宫的邪祟，玉板敲得龙宫的邪雾都散了。

这场大战打了整整三天三夜，龙王的水晶宫都被震裂了，瓦片掉下来砸碎了三个玉杯，龙王只好出面求饶。

三、龙王和解：过海成功

龙王穿着银甲，赔着笑脸从后殿走出来：“各位神仙，是我儿子不懂事，我赔礼了，花篮还你们，还派虾兵蟹将给你们清路。”说着，侍卫就把花篮还给了蓝采和，蓝采和赶紧抱在怀里，检查了半天，连竹篾都没断一根。

吕洞宾摆摆手，酒还没醒：“罢了，下次管好你儿子，别再乱抢东西。”龙王连声应“是”，当场就派了五百虾兵、三百蟹将，在前头清路，把海里的暗礁都移开了。

八仙哈哈大笑，各自用着法宝，顺利过了海。铁拐李的铁拐浮在水上，像艘小舟；汉钟离的芭蕉扇扇得风推着他走，半天就走了百里；张果老的驴倒着走，还驮着剩下的两坛酒；蓝采和的花篮飘在前面，里面还装着刚捞的小海鲜；何仙姑的荷花浮在水面，花瓣上还落了只海鸥；吕洞宾的纯阳剑开路，海浪都分成了两半；韩湘子的箫声飘在前面，引得海豚都跟着跳；曹国舅的玉板平浪，海面平得像镜子。

岸上的老百姓看傻了，有的喊“活神仙”，有的跪下来磕头，这事一传十十传百，就成了千古佳话。

四、千古谚语的由来

“八仙过海，各显神通”的谚语，就是这么来的。老百姓总用这话教育孩子：“你看八仙，出身都不一样，瘸子、倒骑驴的老头、唱跑调的浪子、女流之辈，都能成仙，都能靠自己的本事过海，你们只要好好学本事，啥都能成。”

五、总结

八仙过海的故事，既是他们神通的集中展示，也是“和而不同”的生动写照：八个出身、性格、本事都不同的人，凑在一起，居然能闹出这么大的动静，还留下了千古谚语。提醒我们：不管用什么方法，只要目标一致，就能做成大事；不管多不同，只要心向善，就能成神仙。
